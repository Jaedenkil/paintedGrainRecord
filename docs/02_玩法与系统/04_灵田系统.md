# 灵田系统

> 本章定义灵田的种植、经营与升级机制。灵田是游戏经济循环的基石——产出炼丹材料、烹饪食材、炼器灵木，并通过篆路自动化系统实现全自动生产。

---

## 一、灵田基础

### 1.1 核心概念

灵田拥有 **四项核心参数**，共同决定种植效果：

| 参数 | 说明 | 范围 | 影响 |
|------|------|------|------|
| **土壤品质** | 灵田的基础肥沃度 | 凡土→灵土→灵脉土→仙土 | 生长速度、品质上限 |
| **灵气浓度** | 当前地块的灵气含量 | 0～1000 | 生长速度、变异概率 |
| **湿度** | 土壤含水量 | 0%～100% | 生长速度、枯萎风险 |
| **肥力** | 土壤养分存量 | 0～100（每格独立） | 生长速度、品质下限 |

四项参数中，**土壤品质**是固定属性（升级或改造才改变），其余三项为动态参数，随种植、天气、时间变化。

### 1.2 灵田操作

| 操作 | 前提 | 消耗 | 效果 | 备注 |
|------|------|------|------|------|
| **翻土** | 灵田空闲 | 体力 10 | 重置地块状态，恢复初始参数 | 每季种植前必须执行 |
| **播种** | 已翻土 | 灵种 ×1 | 种植选定灵种 | 可在背包中选择种籽 |
| **浇水** | 已播种 | 灵力 5 或 灵雨符 ×1 | 提升湿度 15%～25% | 过度浇水会烂根 |
| **施肥** | 已播种 | 灵肥 ×1 | 提升肥力 10～20 | 不同灵肥效果不同 |
| **除虫** | 出现害虫 | 触发小游戏 | 清除害虫，恢复健康度 | 失败则品质下降 |
| **除草** | 出现杂草 | 无（手动） | 清除杂草，恢复灵气消耗 | 杂草会抢夺养分 |
| **收获** | 灵植成熟 | 无 | 获得产物 | 延迟收获可能导致品质下降 |
| **升级** | 灵田到达上限等级 | 灵石/材料 | 扩大尺寸/解锁功能 | 见第三章 |
| **聚灵** | 已解锁聚灵阵 | 灵石 ×10/次 | 提升灵气浓度 50～100 | 持续 1 游戏日 |
| **检测** | 无 | 无 | 查看地块详细参数 | 右键地块可查看 |

## 二、灵田属性体系

### 2.1 土壤品质

土壤品质决定灵田的基础性能，通过 **改造** 升级。

| 品质 | 生长速度倍率 | 品质上限 | 解锁条件 | 改造消耗 |
|------|:----------:|:--------:|----------|----------|
| **凡土** | ×1.0 | 良品 | 初始 | — |
| **灵土** | ×1.3 | 极品 | 灵田等级 ≥ 3 | 灵石 ×500 + 基础灵材 ×5 |
| **灵脉土** | ×1.6 | 传说 | 灵田等级 ≥ 5，位于灵脉上方 | 灵石 ×2000 + 中品灵石 ×10 + 灵脉石 ×3 |
| **仙土** | ×2.0 | 传说 | 灵田等级 ≥ 8，完成隐藏任务"九天息壤" | 灵石 ×10000 + 上品灵石 ×5 + 九天息壤 ×1 |

**土壤改造流程**：选择地块 → 消耗材料 → 等待 24h（游戏内）→ 改造完成。改造期间该地块不可种植。

### 2.2 灵气浓度

灵田灵气浓度受以下因素影响：

| 因素 | 影响 | 说明 |
|------|:----:|------|
| 场景基础灵气 | +200～800 | 不同场景灵气浓度不同（灵田默认 500） |
| 聚灵阵 | +50～100/次 | 手动激活，持续 1 游戏日 |
| 灵脉连接 | +300（常驻）| 灵田等级 ≥ 4 后解锁 |
| 种植物消耗 | -1～5/分钟 | 植物生长消耗灵气 |
| 杂草 | -10～20/分钟 | 杂草会大量消耗灵气 |
| 灵植死亡 | -50（一次性）| 枯萎或灾害导致 |
| 天气·灵雨 | +100/小时 | 特殊天气事件 |

**灵气浓度对种植的影响**：
- 浓度 0～200：生长速度 ×0.5，品质上限为凡品
- 浓度 200～500：生长速度 ×1.0，品质上限为良品
- 浓度 500～800：生长速度 ×1.3，品质上限为极品
- 浓度 800～1000：生长速度 ×1.5，品质上限为传说，解锁变异概率

### 2.3 湿度系统

湿度是灵田种植的关键动态参数。

| 湿度区间 | 状态 | 效果 |
|:--------:|------|------|
| 0%～15% | **干旱** | 生长停止，每 5 分钟健康度 -5，持续 30 分钟枯萎 |
| 15%～30% | 偏干 | 生长速度 ×0.7 |
| 30%～70% | **适宜** | 生长速度 ×1.0（最佳区间） |
| 70%～85% | 偏湿 | 生长速度 ×0.9，虫灾概率 +20% |
| 85%～100% | **水涝** | 生长停止，每 5 分钟健康度 -3，持续 30 分钟根腐 |

**湿度自然变化**：
- 晴天：每小时 -5%～-8%
- 阴天：每小时 -2%～-4%
- 雨天：每小时 +10%～+15%
- 灵雨事件：每小时 +20%～+30%

### 2.4 肥力系统

每格灵田拥有独立的肥力值（0～100）。

| 肥力区间 | 效果 |
|:--------:|------|
| 80～100 | 肥力充足，品质下限 +1 档，生长速度 ×1.1 |
| 50～79 | 正常，无修正 |
| 20～49 | 肥力不足，品质下限 -1 档，生长速度 ×0.8 |
| 0～19 | 贫瘠，生长停止，作物缓慢枯萎 |

**肥力消耗**：
- 每株植物每生长周期消耗 10～30 肥力（取决于植物种类）
- 杂草每 10 分钟消耗 2～5 肥力

**灵肥一览**：

| 灵肥 | 效果 | 来源 | 售价 |
|------|------|------|:----:|
| 基础堆肥 | 肥力 +10 | 杂草+厨余合成 | 10 灵石 |
| 骨粉 | 肥力 +15，生长速度 +5%（3天） | 妖兽骨骼研磨 | 25 灵石 |
| 翠灵液 | 肥力 +20，品质 +5% | 炼丹副产物 | 50 灵石 |
| 五行肥 | 肥力 +25，五行亲和 +10（7天） | 万象阁购买 | 200 灵石 |
| 仙酿糟 | 肥力 +30，生长速度 +20%（3天），10%变异概率 | 特殊酿造 | 500 灵石 |

## 三、灵田等级与扩容

### 3.1 升级条件表

| 等级 | 尺寸 | 解锁功能 | 升级消耗 | 种植格数 |
|:---:|:----:|----------|----------|:-------:|
| 1 | 1×1 | 基础种植 | 初始拥有 | 1 |
| 2 | 2×2 | 施肥 | 灵石 ×100 + 木材 ×20 | 4 |
| 3 | 3×3 | 聚灵阵 | 灵石 ×300 + 基础灵材 ×10 | 9 |
| 4 | 4×4 | 灵脉连接、自动灌溉 | 灵石 ×800 + 中品灵石 ×5 + 灵材 ×20 | 16 |
| 5 | 5×5 | 灵田守护阵 | 灵石 ×1500 + 中品灵石 ×10 + 灵材 ×30 | 25 |
| 6 | 6×6 | 五行调和 | 灵石 ×3000 + 上品灵石 ×3 + 五行石 ×5 | 36 |
| 7 | 7×7 | 自动播种（需篆路联动） | 灵石 ×5000 + 上品灵石 ×5 + 灵脉石 ×3 | 49 |
| 8 | 8×8 | 时间加速阵 | 灵石 ×8000 + 极品灵石 ×3 + 九天息壤 ×1 | 64 |
| 9 | 9×9 | 自动收获（需篆路联动） | 灵石 ×12000 + 极品灵石 ×5 + 道蕴石 ×1 | 81 |
| 10 | 10×10 | 灵田之灵（器灵） | 灵石 ×20000 + 传说灵材 ×3 + 魂晶 ×1 + 特殊任务 | 100 |

> **设计意图**：前 5 级为经营养成核心内容，大部分玩家可达到；6～8 级为后期深度内容；9～10 级为终极目标级内容。

### 3.2 特殊功能详解

**聚灵阵（Lv.3）**：手动激活，消耗灵石 ×10，提升灵田灵气浓度 +50～100，持续 1 游戏日。可与灵脉连接叠加。

**自动灌溉（Lv.4）**：每日自动检测湿度，若低于 30% 则自动补充至 50%。消耗灵石 ×5/次。需预先在灵田控制面板设置。

**灵脉连接（Lv.4）**：将灵田连接到地底灵脉，常驻灵气 +300。需场景下方存在灵脉方块。若灵脉枯竭则效果减半。

**灵田守护阵（Lv.5）**：降低灾害概率 50%，自动抵御低级妖兽侵扰。消耗中品灵石 ×1/月（游戏内）。

**五行调和（Lv.6）**：解锁五行轮作系统。每种灵植消耗对应五行灵气，自动平衡避免单一属性枯竭。五行平衡时全生长速度 +15%。

**自动播种（Lv.7）**：配合篆路自动化系统，设定种子类型后自动播种种籽。需连接种籽箱（Inventory 接入）。

**时间加速阵（Lv.8）**：灵田内时间流速 ×2（现实时间不变），持续 2 小时/次，冷却 12 小时。消耗极品灵石 ×1/次。

**自动收获（Lv.9）**：配合篆路自动化系统，灵植成熟后自动收割并存入仓库。需连接仓库容器。

**灵田之灵（Lv.10）**：灵田获得器灵，可自动管理一切日常操作（浇水/施肥/除虫/收获），定期与玩家对话，提示灵田状态。器灵可命名、可互动，是隐藏结局"天人合一"的触发条件之一。

## 五、种植机制详解

### 5.1 生长周期公式

```
实际生长时间 = 基础生长时间 × 土壤品质系数 × 灵气系数 × 湿度系数 × 肥力系数 × 特殊修正
```

**系数参考**：

| 系数 | 条件 | 倍率 |
|------|:----:|:----:|
| 土壤品质 | 凡土/灵土/灵脉土/仙土 | 1.0 / 0.8 / 0.6 / 0.5 |
| 灵气浓度 | ＜200 / 200～500 / 500～800 / 800+ | 2.0 / 1.0 / 0.75 / 0.6 |
| 湿度 | 干旱/偏干/适宜/偏湿/水涝 | 生长停止 / 1.5 / 1.0 / 1.2 / 生长停止 |
| 肥力 | 0～19 / 20～49 / 50～79 / 80+ | 生长停止 / 1.3 / 1.0 / 0.85 |
| 时间加速阵 | 激活 | ×0.5 |
| 灵田守护阵 | 激活 | ×0.95 |
| 五行调和 | 平衡 | ×0.85 |

**示例**：在灵土（×0.8）+ 灵气 600（×0.75）+ 适宜湿度（×1.0）+ 肥力 85（×0.85）条件下种植凝露草：
`5 min × 0.8 × 0.75 × 1.0 × 0.85 = 2.55 min`（约 2 分 33 秒）

### 5.2 品质判定

灵植最终品质由 **种籽品质 × 土壤品质 × 照料评分 × 随机因子** 决定。

**种籽品质**（初始决定品质上限）：
| 种籽品质 | 来源 | 品质上限 |
|:--------:|------|:--------:|
| 普通种籽 | 杂货铺购买 | 良品 |
| 良种种籽 | 万象阁/任务奖励 | 极品 |
| 极品种籽 | 特殊掉落/杂交 | 传说 |
| 传说种籽 | 隐藏任务/变异 | 传说（高概率）|

**照料评分**（0～100，影响最终品质档位的浮动）：
| 操作 | 加分 | 扣分 |
|------|:----:|:----:|
| 浇水（适宜）| +2/次 | — |
| 浇水（过度）| — | -5/次 |
| 施肥 | +5/次 | — |
| 成功除虫 | +10/次 | — |
| 除虫失败 | — | -15/次 |
| 除草 | +3/次 | — |
| 干旱未处理 | — | -2/分钟 |
| 水涝未处理 | — | -2/分钟 |

**最终品质判定**（照料评分 ≥ 60 时）：
```
Roll = 随机(0, 100) + 照料评分 × 0.3
品质 = 对照种籽品质上限 + Roll 偏移
```

| Roll 区间 | 品质 |
|:---------:|:----:|
| 0～20 | 降 1 档 |
| 21～60 | 不变 |
| 61～85 | 升 1 档 |
| 86～100 | 升 2 档（不超过上限）|

### 5.3 杂交与变异

**杂交系统**：当相邻地块（上下左右）种植不同灵植时，收获时有概率获得杂交种籽。

```
杂交概率 = 5%（基础）+ 灵气浓度加成 + 五行相生加成
杂交种籽 = 母本种类（新）+ 父本特性（继承 1～2 项属性）
```

**杂交示例**：
- 凝露草 + 火灵花 → **火凝草**（生长 10min，炼制火系回灵丹）
- 金线莲 + 紫芝 → **金纹紫芝**（品质上限 +1 档，炼制极品筑基丹）
- 朱果 + 蕴灵果 → **朱蕴果**（恢复 + 修为双效果）

**变异系统**：灵植成熟时有 0.5%～5%（取决于灵气浓度）概率发生变异。

| 变异类型 | 触发条件 | 效果 |
|:--------:|----------|------|
| 体型变异 | 灵气 800+ | 产量 ×2～3 |
| 属性变异 | 五行偏向 | 附加五行伤害/抗性效果 |
| 颜色变异 | 随机 | 收藏价值，可在万象阁高价出售 |
| 灵智变异 | 灵气 1000+ + 传说种籽 | 诞生灵植宠（可对话、辅助战斗） |

### 5.4 灾害系统

灵田可能遭遇以下灾害，需玩家及时应对：

| 灾害 | 触发条件 | 效果 | 应对方式 | 损失 |
|:----:|----------|------|----------|:----:|
| **虫灾** | 湿度过高(>70%) + 随机 | 灵植健康度 -10/分钟 | 除虫小游戏（点击害虫） | 轻度：减产 20% |
| **枯萎病** | 肥力 <20 + 持续干旱 | 灵植健康度 -15/分钟 | 施肥 + 浇水 + 灵药治疗 | 中度：减产 50% |
| **灵气风暴** | 特殊天气事件 | 灵植连根拔起 30%概率 | 开启守护阵（Lv.5） | 重度：损失 30%～80% |
| **寒潮** | 季节系统（冬季） | 生长停止，非耐寒植物受损 10%/h | 建造暖棚（建筑 Lv.3） | 中度：生长延迟 |
| **妖兽侵扰** | 灵田等级高 + 未设守护阵 | 随机吞噬 1～3 株成熟灵植 | 手动驱赶 / 守护阵 / 陷阱 | 重度：损失成熟作物 |

**灾害预警系统**：
- 灵田等级 ≥ 3：提前 1 天预警灵气风暴
- 灵田等级 ≥ 5：守护阵自动侦测妖兽靠近
- 灵田等级 ≥ 7：提前 3 天预警寒潮
- 天道参数·七星：影响天象类灾害的频率

## 六、与篆路自动化系统的联动

灵田是篆路自动化系统的核心应用场景之一。参考 [`14_篆路自动化系统`](02_玩法与系统/14_篆路自动化系统.md) 的设计原则，灵田自动化需铺设相应的篆路网络。

### 6.1 可自动化操作

| 操作 | 所需篆路组件 | 对应篆文阶位 |
|------|-------------|:-----------:|
| 自动浇水 | 湿度传感器 + 浇水执行器 | 第五阶·灵变 |
| 自动施肥 | 肥力传感器 + 施肥执行器 + 灵肥箱 | 第四阶·通幽 |
| 自动播种 | 种籽箱 + 播种执行器 + 空地块检测 | 第三阶·玄妙 |
| 自动收获 | 成熟检测传感器 + 收获执行器 + 仓库 | 第三阶·玄妙 |
| 条件除虫 | 害虫检测传感器 + 除虫法器 | 第四阶·通幽 |
| 智能轮作 | 五行检测 + 种籽切换器 + 记录模块 | 第二阶·造化 |

### 6.2 典型自动化配置

**初级自动化（灵田 Lv.4 可解锁）**：
```
[湿度传感器] → [对比器(＜30%)] → [浇水执行器]
```
效果：湿度低于 30% 时自动浇水至 50%。成本：篆路组件 ×3。

**中级自动化（灵田 Lv.5～7）**：
```
[湿度传感器] → [对比器] → [浇水执行器]
[肥力传感器] → [对比器(＜50)] → [施肥执行器] ← [灵肥箱]
[成熟传感器] → [收获执行器] → [仓库容器]
```
效果：全自动浇水 + 补肥 + 收获。成本：篆路组件 ×8。

**高级自动化（灵田 Lv.7～9）**：
```
[种籽箱] → [播种执行器] ← [空地块检测]
[湿度/肥力/灵气传感器] → [中央处理器] → [各执行器]
[成熟传感器] → [收获执行器] → [分拣器] → [仓库/炼丹炉]
[五行检测器] → [轮作控制器] → [种籽切换器]
```
效果：全自动种植流水线，玩家只需补充种籽和灵肥。成本：篆路组件 ×20+。

### 6.3 限制条件

- 自动化篆路需要消耗灵石作为驱动能源（1 灵石/组件/游戏日）
- 每个执行器每日有操作次数上限（取决于组件品质）
- 篆路组件需要定期维护（维护成本 = 组件数 × 10 灵石/周）
- 自动化无法替代高品质手动照料——全自动种植的品质上限锁定在良品

### 7.3 高阶策略：品质优先 vs 产量优先

| 策略 | 核心思路 | 适用阶段 | 推荐灵植 |
|:----:|----------|:--------:|----------|

| **产量优先** | 快速周转，薄利多销 | 炼气～筑基 | 凝露草、玉芽米、青玉莓 |
| **杂交育种** | 投入时间培育新品种，长期回报 | 金丹+ | 相邻异种搭配 |
| **自动化躺平** | 投入篆路建设，解放双手 | 元婴+ | 任意高价值作物 |
| **五行平衡** | 五行属性均衡种植，触发全局加成 | 金丹+ | 五行各一种搭配 |
