# 核心循环

> 本章定义玩家在《云汲仙田录》中的日常与战斗循环。

---

## 日常循环（经营侧）

```
起床 → 检查灵田（浇水/除虫/收获）
       ↓
    炼制丹药 / 法器
       ↓
    接取任务 / 事件
       ↓
    出发探索 / 战斗
       ↓
    返回 → 整理战利品
       ↓
    修炼 → 尝试突破
       ↓
    保存 → 休息
```

---

## 战斗循环（探索侧）

```
进入区域 → 遭遇敌人
          ↓
    【起手】绘制符箓 → 远程压制
          ↓
    【中盘】古篆文激活 → 法阵/ Buff / 控场
          ↓
    【近战】剑诀 / 身法 → 收割
          ↓
    【收尾】处决 → 额外掉落
          ↓
     战利品结算 → 受伤评估 → 继续 / 撤退
```

---

## 时间尺度

| 周期 | 玩家行为 | 系统反馈 |
|------|----------|----------|
| 秒级 | 闪避、攻击、绘制符箓 | 伤害数字、状态效果 |
| 分钟级 | 清理一个区域/副本 | 掉落结算、经验获取 |
| 小时级 | 完成一次探索循环 | 灵田收获、炼成一炉丹 |
| 天级 | 境界突破、制作高级装备 | 新区域解锁、新配方 |
| 周/月级 | 主线章节推进 | 新系统解锁、剧情转折 |
