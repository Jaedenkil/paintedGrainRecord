# 炼丹炼器

> 本章定义合成系统的核心机制，包括炼丹（消耗品产出）与炼器（装备产出）两大分支。底层数值运算（九宫炼丹/五行生克/八卦相荡）由 [`17_天道参数化系统`](02_玩法与系统/17_天道参数化系统.md) 承载，本章仅定义玩家可感知的配方、品质、流程与交互。

---

## 一、炼丹系统

### 1.1 炼丹流程

```
收集/学习丹方 → 备齐原料 → 选择丹炉 → 调节火候（小游戏） → 成丹判定
     ↑                                                           |
     └────────────── 失败：返还部分材料 ←———————————————┘
```

| 步骤 | 说明 |
|------|------|
| **丹方获取** | 主线奖励、万象阁购买、秘境掉落、NPC 传授 |
| **原料准备** | 按丹方所需的药材/辅料，从背包放入丹炉 |
| **丹炉选择** | 不同丹炉影响成功率、出丹数量、品质上限 |
| **火候控制** | 节奏小游戏（详见 1.5），直接影响最终品质 |
| **成丹判定** | 基础成功率 × 炉具修正 × 材料品质修正 × 小游戏得分修正 |

### 1.2 丹炉系统

| 丹炉 | 需求境界 | 基础成功率 | 品质上限 | 成丹数量 | 获取方式 |
|------|---------|-----------|---------|---------|---------|
| 陶土炉 | 凡人 | +0% | 良品 | 1~2 | 初始赠送 |
| 青铜炉 | 炼气 | +5% | 极品 | 1~3 | 万象阁（灵石 ×800） |
| 玄铁炉 | 筑基 | +10% | 极品 | 2~3 | 炼器锻造 / 任务奖励 |
| 紫金丹炉 | 金丹 | +15% | 传说 | 2~4 | Boss「炎火道人」掉落 |
| 天工炉 | 元婴 | +20% | 传说 | 3~5 | 暗影楼任务链奖励 |
| 混元炉 | 化神 | +25% | 传说 | 3~6 | 九宫炼丹完美品质奖励 |

> 丹炉可在锻造台升级（+1~+3），每级 +3% 成功率、+1 格九宫炼丹槽位。

### 1.3 丹药品级判定

| 品级 | 基础概率 | 成丹数量 | 效果倍率 | 额外词条 |
|------|---------|----------|---------|---------|
| 凡品 | 45% | 1~2 | ×1.0 | 无 |
| 良品 | 30% | 2~3 | ×1.5 | 无 |
| 极品 | 18% | 3~4 | ×2.0 | 概率附带 1 条随机附加效果 |
| 传说 | 7% | 4~5 | ×3.0 | 必定附带 1~2 条随机附加效果 |

**品质判定流程**：
```
最终品质概率 = 基础概率 × 炉具品质修正 × 材料品质修正 × 小游戏得分修正
判定方式：Roll 点（0~100）
  • 0~7  → 传说（若品质上限允许）
  • 8~25 → 极品
  • 26~55 → 良品
  • 56~100 → 凡品
```

> 在九宫炼丹模式下，品质判定替换为[`17_天道参数化系统·第5章`](02_玩法与系统/17_天道参数化系统.md) 的九宫合丹规则。

### 1.5 控火小游戏

**玩法**：炉火强度以进度条显示，火候需在"最佳区间"内维持 5 秒。

```
  过低 ← [████████░░░░░░░░] → 过高
           ↑最佳区间
```

| 参数 | 值 |
|------|-----|
| 游戏时长 | 8s（凡品丹炉）→ 5s（极品丹炉） |
| 最佳区间宽度 | ±15%（凡品）→ ±25%（传说炉） |
| 操作频率 | 每 0.5s 可通过按键/点击调整火力（↑增强/↓减弱） |
| 得分计算 | 处于最佳区间的时间占比 × 100% |
| 判定 | 得分 ≥ 80% 时品质 +1 档（不超过上限）；得分 ≤ 30% 时品质 -1 档 |

> 小游戏可通过`07_界面设计/04_系统设置.md`中的「跳过小游戏」开关关闭（关闭后默认 60% 得分）。

### 1.6 成功率详细公式

```
最终成功率 = 基础成功率（45%） 
           + 丹炉修正（陶土±0 / 青铜+5% / 玄铁+10% / …）
           + 材料品质修正（普通±0 / 良品+5% / 极品+10%）
           × 小游戏得分修正（得分% × 0.3，最高 +21%）
           + 天道参数修正（九宫炼丹模式下替换为九宫规则）

失败时返还：主材料 ×50% / 辅材料 ×30% / 丹炉不损耗
```

### 1.7 天道参数化炼丹（引用）

[`17_天道参数化系统·第5章`](02_玩法与系统/17_天道参数化系统.md) 定义了高阶炼丹模式——**九宫炼丹**：

- 使用九宫格界面放置 9 份材料（中宫定主、坎宫传导、离宫催化）
- 材料的天道指纹（五行/八卦/九宫值）按九宫规则融合
- 七星择时：七曜日对特定丹药类型有加成
- 产出丹药的品质和附加属性由天道指纹计算，而非Roll点
- 是本配方系统的**上层扩展**，玩家达到金丹期后解锁

## 二、炼器系统

### 2.1 炼器流程

```
获得/学习图纸 → 收集材料 → 选择锻造台 → 锤炼（小游戏） → 成器判定
      ↑                                                      |
      └────────── 失败：材料损耗 ————————————————┘
```

| 步骤 | 说明 |
|------|------|
| **图纸获取** | Boss 掉落、宗门贡献兑换、秘境探索、主线奖励 |
| **材料准备** | 按图纸要求放入主材料 + 辅材料（影响词条） |
| **锻造台选择** | 不同锻造台影响品质上限、词条数量、成功率 |
| **锤炼小游戏** | 节奏连击小游戏（详见 2.5） |
| **成器判定** | 品质判定 + 词条数量判定 + 天道参数附加效果 |

### 2.2 锻造台系统

| 锻造台 | 需求境界 | 基础成功率 | 品质上限 | 词条下限 | 获取方式 |
|--------|---------|-----------|---------|---------|---------|
| 石砧 | 凡人 | +0% | 良品 | 0 | 初始赠送 |
| 铁砧 | 炼气 | +5% | 极品 | 1 | 万象阁（灵石 ×1000） |
| 精金砧 | 筑基 | +10% | 极品 | 1 | 宗门贡献兑换 / 炼器专精 |
| 灵锻台 | 金丹 | +15% | 传说 | 2 | Boss「熔岩巨人」掉落 |
| 天工台 | 元婴 | +20% | 传说 | 2 | 暗影楼任务链奖励 |
| 造化台 | 化神 | +25% | 传说 | 3 | 九宫锻造完美品质解锁 |

### 2.3 装备品质体系

| 品质 | 基础概率 | 词条数 | 基础属性倍率 | 特殊 |
|------|---------|--------|------------|------|
| 凡品 | 40% | 0~1 | ×1.0 | 无 |
| 良品 | 30% | 1~2 | ×1.3 | 无 |

### 2.5 锤炼小游戏

**玩法**：在节奏提示到达判定区时按下对应按键，连续成功可积累连击倍率。

```
  ┌──────●──────┐
  │      ●      │  ← 判定区（绿色为完美）
  │  ●   ●   ●  │  ← 节奏提示依次下落
  │ ● ● ● ● ● │
  └──────────────┘
```

| 参数 | 值 |
|------|-----|
| 节奏数 | 8 次击打（凡品台）→ 5 次（传说台） |
| 判定精度 | ±150ms（完美）/ ±300ms（通过）/ 超出=失误 |
| 连击奖励 | 每连续成功 1 次 +5% 品质概率、最多 +25% |
| 失误惩罚 | 连击清零，品质概率 -10% |

> 同样可通过「跳过小游戏」开关跳过（默认 60% 得分）。

### 2.6 天道参数化锻造（引用）

[`17_天道参数化系统·第4章`](02_玩法与系统/17_天道参数化系统.md) 定义了高阶锻造模式：

- **五行生克矩阵**：辅材料对主材料有 5×5 加权修正（相克 ×0.75、相生 ×1.5）
- **八卦相荡**：主辅材料八卦值的 8×8 组合对应表，28 种有效组合产生特殊锻造效果
- **场景指纹继承**：材料携带采集场景的天道指纹传递至成品
- 是本文本配方系统的**上层扩展**，玩家达到金丹期后解锁

## 三、材料系统

### 3.1 材料分类

| 大分类 | 子类 | 示例 | 主要获取 |
|--------|------|------|---------|
| **药材** | 草本/花/果/根 | 凝露草、金线莲、蕴灵果 | 采集/灵田种植 |
| **矿石** | 普通/稀有/灵矿 | 铁矿石、灵石、玉髓 | 采矿/战斗掉落 |
| **灵材** | 五行属性 | 金晶石、炎火鳞、寒玉 | 特定场景采集/Boss 掉落 |
| **兽材** | 普通/精英/Boss | 兽骨粉、蛟龙内丹、凤凰羽 | 战斗掉落 |
| **辅料** | 消耗品 | 清水、灵石碎、灵泉水 | 杂货铺购买/环境采集 |
| **特殊** | 任务/剧情 | 劫雷木、天道碎片、道果 | 任务奖励/隐藏探索 |

### 3.2 材料品质

| 品质 | 效果 | 获取概率 |
|------|------|---------|
| 普通（白色） | 基础效果 ×1.0 | 60% |
| 良品（绿色） | 基础效果 ×1.3 | 25% |
| 极品（蓝色） | 基础效果 ×1.6 | 12% |
