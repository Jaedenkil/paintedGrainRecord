# 新手引导系统

> 本章定义游戏新手引导的设计原则、解锁顺序和交互方式。目标是在不打断玩家沉浸感的前提下，渐进式地教会玩家所有核心系统。

---

## 一、引导设计原则

### 1.1 核心原则

| 原则 | 说明 |
|:----:|------|
| **渐进式解锁** | 不一次性展示所有系统。按玩家进度逐步开放新功能和 UI 元素 |
| **非强制可跳过** | 基础移动/战斗教学强制（序章），其余引导均可跳过或关闭 |
| **可重复查看** | 所有教学引导均可通过「修炼手札」重新查看 |
| **不打断战斗** | 战斗期间不会弹出引导提示，待战斗结束后再触发 |
| **情境触发** | 引导不在无关联时弹出，只在玩家首次遇到相关系统时触发 |
| **正向反馈** | 每完成一个引导步骤给予小奖励（灵石/经验），强化学习动机 |

### 1.2 引导分级

| 级别 | 强制程度 | 说明 | 示例 |
|:----:|:--------:|------|:----:|
| **S级** | 强制 | 核心操作，不完成无法继续 | 移动、基础攻击、灵田种植 |
| **A级** | 推荐 | 重要系统，弹出提示但可关闭 | 炼丹炼器、技能点分配、装备 |
| **B级** | 可选 | 进阶内容，仅提示入口 | 篆路自动化、阵法、古篆收集 |
| **C级** | 隐藏 | 不主动提示，留待玩家自行发现 | 杂交育种、隐藏任务、天道参数 |

## 二、序章引导流程

> 序章「梦入仙途」是游戏的第一个关卡/场景，玩家在此完成全部 S 级引导。

### 2.1 分步引导总表

| 步骤 | 教学内容 | 触发时机 | 交互方式 | 奖励 |
|:----:|:--------:|----------|:--------:|:----:|
| **① 初始苏醒** | 视角控制、移动（WASD/方向键）| 游戏开始 | 高亮移动键 → 强制移动至目标点 | — |
| **② 交互操作** | 按 E/空格与 NPC 对话 | 接近青竹老人 | 高亮交互键 + 对话气泡 | 凝露草种籽 ×3 |
| **③ 灵田种植** | 翻土→播种→浇水→等待→收获 | 获得种籽后 | 分步高亮操作按钮 + 禁止跳步 | 灵石 ×10 |
| **④ 基础战斗** | 攻击（左键/确定键）、闪避（Shift）| 妖兽侵扰事件 | 高亮攻击键 + 强制击败竹叶青 ×2 | 灵石 ×20 |
| **⑤ 灵力使用** | 释放符箓（Q 键切换，右键释放）| 击败竹叶青后 | 高亮技能栏 + 指引释放 | 引火符 ×3 |
| **⑥ 境界突破** | 打开修炼面板 → 点击突破 | 灵力满后自动提示 | 高亮修炼面板按钮 | 境界突破奖励 |
| **⑦ 基础采集** | 采集矿石/药草（走到附近按 E）| 前往落霞山脉 | 高亮可采集物 + 引导采集 | 基础材料 ×3 |
| **⑧ 第一次炼丹** | 打开丹炉 → 放入材料 → 点火 | 采集到药材后 | 高亮配方选择 + 引导操作 | 回灵丹 ×3 |
| **⑨ 技能学习** | 打开技能树 → 分配技能点 | 突破至炼气2层 | 高亮技能树 + 推荐加点 | 技能点 +1 |

### 2.2 序章教学关设计

**场景**：「梦入仙途」—— 一个简化的灵田 + 山谷地图，安全区域和战斗区域分界明显。

| 区域 | 用途 | 限制 |
|:----:|:------|:----:|
| **灵田教学区** | 引导 ①～③ | 无敌状态，无法受到伤害 |
| **战斗教学区** | 引导 ④～⑤ | 竹叶青 ×2，属性降低 50% |
| **修炼教学区** | 引导 ⑥ | 自动满灵力状态 |
| **自由练习区** | 综合演练 | 无限制，可随时离开 |

**序章完成条件**：完成以上全部 9 个引导步骤，并击败竹叶青 ×2。

**序章奖励**（完成全部引导后）：
- 灵石 ×50
- 凝露草种籽 ×5
- 回灵丹 ×5
- 基础符箓 ×3（随机属性）
- 称号「初入仙途」

## 三、系统解锁顺序

### 3.1 按玩家进度解锁

| 玩家条件 | 解锁系统 | 引导级别 | 引导内容 |
|:--------:|:---------|:--------:|----------|
| 游戏开始 | 移动/交互/灵田基础 | S | 强制教学关 |
| 击败第一个敌人 | 基础战斗/符箓 | S | 强制引导 |
| 突破至炼气 | 修为面板/技能树 | A | 弹出提示 |
| 首次采集材料 | 背包/物品栏 | A | 高亮背包按钮 |
| 炼气3层 | 炼丹炼器系统 | A | 引导第一次炼丹 |
| 炼气5层 | 装备系统/强化 | A | 弹出装备界面提示 |
| 筑基期 | 古篆文收集/镶嵌 | B | 提示「发现神秘篆文」|
| 筑基3层 | 经脉系统 | B | 弹出经脉界面引导 |
| 筑基5层 | 阵法系统 | B | 提示阵法入口 |
| 金丹期 | 篆路自动化 | B | 首次获得灵路组件时引导 |
| 金丹3层 | 法宝系统 | A | 弹出法宝槽位提示 |
| 金丹5层 | 拍卖行 | B | 首次进入万象阁时引导 |
| 元婴期 | 天道参数化 | C | 无主动引导 |
| 元婴3层 | 杂交培育 | C | 首次相邻种植时概率提示 |
| 化神期 | 双模板系统 | C | 无主动引导 |

### 3.2 系统解锁可视化

建议在游戏 UI 中使用 **系统解锁进度图**（类似技能树），展示已解锁/待解锁的系统：

```
[灵田] ✓ → [战斗] ✓ → [炼丹] ✓ → [符箓] ✓ → [技能树] ✓
                                                ↓
[装备] ✓ ← [古篆] 🔒 ← [经脉] 🔒 ← [阵法] 🔒 ← [筑基突破]
                                                ↓
[法宝] 🔒 ← [篆路] 🔒 ← [拍卖] 🔒 ← [杂交] 🔒 ← [天参数] (隐藏)
```

## 四、引导交互方式

### 4.1 四种引导类型

| 类型 | 适用场景 | 实现方式 | 可跳过 |
|:----:|:---------|:---------|:------:|
| **高亮指引** | 按钮位置提示 | 目标 UI 元素发光 + 外围暗色遮罩 + 箭头指向 | 是 |
| **气泡提示** | 系统说明/提示 | NPC 对话气泡或 UI 右上角弹出信息框 | 是 |
| **强制教学** | S 级操作 | 锁定其他操作，分步指导，必须完成当前步骤 | 否 |
| **提示面板** | 系统解锁 | 半透明面板展示新系统概览 + "知道了"按钮 | 是 |

### 4.2 高亮指引规范

```
高亮颜色: #FFD700 (金色) — 引导目标
遮罩颜色: rgba(0, 0, 0, 0.6) — 非目标区域
箭头动画: 从屏幕外指向目标，持续 3s 后淡出
文字提示: 目标元素上方或下方，白底黑字 + 图标
```

**高亮指引示例**（灵田种植）：
1. 遮罩全屏，仅「翻土」按钮高亮
2. 箭头指向翻土按钮，文字提示"点击翻土，准备种植"
3. 玩家点击翻土 → 遮罩更新，高亮「种籽栏」
4. 玩家选择种籽 → 高亮「播种」按钮
5. 以此类推，直到收获完成

### 4.3 气泡提示规范

```
位置: 屏幕右上角（系统提示）/ NPC 头顶（对话提示）
最大宽度: 400px
显示时间: 自动 8s 后消失，或玩家点击关闭
图标: 左侧显示对应系统的 Icon
```

**气泡提示示例**（首次获得装备）：
```
┌─────────────────────────────┐
│ [剑 Icon] 获得新装备：竹剑   │
│ 打开背包查看装备属性 (按 B)   │
│                    [× 关闭]  │
└─────────────────────────────┘
```

### 4.4 提示面板规范（系统解锁）

```
┌─────────────────────────────────────────┐
│  🎉 解锁新系统：炼丹炼器                  │
│                                          │
│  [丹炉 Icon] 炼丹炼器系统已解锁           │
│  你可以使用采集到的材料炼制丹药和装备。     │
│                                          │
│  📖 修炼手札 → 炼丹篇 可查看详细教程       │
│                                          │
│              [知道了]  [前往炼丹]          │
└─────────────────────────────────────────┘
```

## 五、教学关设计

### 5.1 序章教学关「梦入仙途」

**场景规格**：
- 尺寸：20×20 方块（2.5D 视角）
- 包含：灵田 2×2、竹屋 ×1、战斗区域 10×10、传送门 ×1
- 音乐：轻柔的竹林风声 + 古筝背景

**NPC**：青竹老人（全程引导 NPC）
- 不可跳过对话（仅第一次）
- 对话中包含游戏世界观介绍
- 完成引导后赠送初始资源和灵田种籽

**战斗教学**：
- 敌人：竹叶青 ×2（属性降低 50%，攻击力 6）
- 目标：击败两只竹叶青
- 提示：第一次被攻击时自动触发闪避教学
- 安全机制：HP 低于 20% 时自动恢复至 50%

### 5.2 后续系统教学关

| 系统 | 教学关形式 | 触发条件 |
|:----:|:----------|:---------|
| 炼丹 | 青竹老人炼丹教学（NPC 对话 + 引导操作）| 首次采集到药材 |
| 符箓 | 灵符谷 NPC 示范绘制 | 首次获得符笔 |
| 古篆文 | 发现第一个古篆碑文时的沉浸式教学 | 首次进入遗迹场景 |
| 篆路自动化 | 灵田升级 Lv.4 时的组件安装引导 | 首次获得篆路组件 |
| 阵法 | 宗门阵法堂的入门教学 | 首次加入宗门后 |

## 六、可重放教程系统：「修炼手札」

### 6.1 系统概述

「修炼手札」是游戏内置的教程百科，玩家可随时按 `H` 键打开。

**设计目标**：玩家不需要去 Wiki 查资料，所有教学内容都在游戏中可查。

### 6.2 手札分类

| 卷册 | 内容 | 包含条目 |
|:----:|:------|:--------:|
| **基础篇** | 界面操作、移动、交互 | 12 条 |
| **灵田篇** | 种植、升级、自动化 | 8 条 |
| **战斗篇** | 攻击、符箓、剑诀、闪避 | 10 条 |
| **修炼篇** | 境界突破、技能树、经脉 | 8 条 |
| **炼丹炼器篇** | 炼丹流程、炼器流程、品质体系 | 10 条 |
| **古篆篇** | 收集、镶嵌、符文套装 | 6 条 |
| **经济篇** | 货币、商铺、拍卖、强化 | 8 条 |
| **进阶篇** | 篆路自动化、阵法、天道参数 | 6 条 |

### 6.3 手札条目格式

每一条目包含：
- **标题**：简短明了
- **内容**：2～5 段文字说明 + 关键操作提示
- **示意图**：界面截图或操作示意图（像素风格）
- **相关链接**：引用其他系统文档的交叉链接
- **视频演示按钮**：（可选）播放录制的操作演示

**示例条目**（灵田种植）：
```
┌─── 灵田种植 ─────────────────────────┐
│                                       │
│ 1. 翻土：在空闲灵田上点击「翻土」      │
│ 2. 播种：选择背包中的灵种种籽          │
│ 3. 浇水：灵力消耗 5，湿度提升 20%      │
│ 4. 等待：生长进度条满后可收获           │
│                                       │
│ 💡 提示：不同灵植的生长时间和条件不同   │
│ 📖 参考：04_灵田系统.md                │
│                                       │
│           [播放演示]  [知道了]          │
└───────────────────────────────────────┘
```

  "trigger": {
    "type": "first_action",
    "action": "enter_scene",

  "steps": [
    {
      "type": "highlight",
