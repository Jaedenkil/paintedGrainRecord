# 阵法系统

> 本章详细定义阵法的学习、布置、激活与组合机制。

---

## 系统概述

阵法是《云汲仙田录》中高阶修士运用的 **领域级能力**。与符箓（即时释放）和古篆文（装备镶嵌）不同，阵法需要 **在场景中预先布置**，激活后形成一片持续存在的 **法则领域**，为范围内的友方提供增益、对敌方施加减益或造成持续伤害。

阵法系统与 **古篆文系统** 深度关联——古篆文是阵法的"字符"，阵法是古篆文的"语法"。

## 阵法类型

### 按用途分类

| 类型 | 说明 | 示例 |
|------|------|------|
| **杀阵** | 对进入范围内的敌人造成伤害 | 天罡剑阵、万剑归宗阵 |
| **困阵** | 限制敌人移动/行动 | 八卦迷踪阵、缚灵阵 |
| **护阵** | 为友方提供防御/回复 | 四象守护阵、回春阵 |
| **辅助阵** | 提供增益效果 | 聚灵阵、疾风阵、金刚阵 |
| **特殊阵** | 特殊功能 | 传送阵、隐匿阵、时间阵 |

### 按规模分类

| 规模 | 覆盖范围 | 布置时间 | 灵力消耗 | 持续 |
|------|----------|----------|----------|------|
| **微型** | 5×5 tiles | 1s | 低 | 10s |
| **小型** | 10×10 tiles | 2s | 中 | 30s |
| **中型** | 20×20 tiles | 3s | 高 | 60s |
| **大型** | 40×40 tiles | 5s | 极高 | 120s |
| **巨型** | 全场景 | 剧情触发 | — | 剧情限定 |

## 阵法与古篆文的联动

阵法是古篆文的 **高级应用**。布置阵法需要将多个古篆文按特定 **阵纹排列** 组合：

### 阵纹结构

```
               ☰ (天) - 阵眼
              /   \
          ☲ (火)  ☵ (水) - 阵基
            \     /
         ☳ (雷)  ☴ (风) - 阵脚
```

- **阵眼**：阵法的核心，决定阵法类型
- **阵基**：支撑阵法的基石，决定阵法强度和持续时间
- **阵脚**：阵法的辅助结构，决定附加效果

### 阵法组合示例

| 阵法名称 | 阵眼需求 | 阵基需求 | 阵脚需求 | 效果 |
|----------|----------|----------|----------|------|
| **三才聚灵阵** | 天 | 地 + 人 | — | 范围内灵力恢复 +200% |
| **四象杀阵** | 火 | 金 + 木 + 水 | — | 范围内敌人持续受到四属性伤害 |
| **五行困阵** | 土 | 金 + 木 + 水 + 火 | — | 范围内敌人减速 50% + 定身概率 |
| **天罡北斗阵** | 天 + 地 | 水 + 火 | 雷 + 风 | 全属性大幅提升，免疫控制 |
| **混沌灭世阵** | 所有古篆文 | — | — | 全屏巨额伤害（剧情限定） |

## 布置与激活

### 布置流程

```
选择阵法 → 消耗对应古篆文（作为阵纹标记）
   ↓
进入布置模式（俯视视角/网格预览）
   ↓
选择阵眼位置 → 依次放置阵基 → 阵脚
   ↓
灵力注入 → 阵法激活
```

### 激活条件

- **灵力充足**：消耗与阵法规模成正比
- **古篆文就绪**：对应古篆文已收集并处于可用状态
- **境界达标**：玩家境界不低于阵法等级要求
- **地形适宜**：部分阵法需要特定地形（如水域阵法需在水面）

### 阵法维护

- 激活后持续消耗 **灵力**（可通过灵石/丹药补充）
- 受到攻击会加速灵力消耗
- 阵眼被破坏 → 阵法提前失效
- 可手动解除 → 回收部分古篆文

## 阵法专精

玩家可通过 **阵法熟练度** 提升专精等级：

| 专精等级 | 效果 |
|----------|------|
| 初窥门径 | 基础阵法解锁 |
| 略有所成 | 布置速度 +20%，灵力消耗 -10% |
| 融会贯通 | 可同时维持 2 个阵法 |
| 出神入化 | 可布置变形/组合阵法（两个阵法融合） |
| 道法自然 | 可同时维持 3 个阵法，灵力消耗 -50% |
