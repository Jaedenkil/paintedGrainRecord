# 战斗系统

> 本章详细定义《云汲仙田录》的战斗机制，包括完整的属性体系、伤害公式与状态效果。

---

## 战斗模式总览

《云汲仙田录》采用 **实时 2D 动作战斗**，玩家可在三种战斗方式间自由切换：

| 方式 | 定位 | 操作方式 | 资源消耗 |
|------|------|----------|----------|
| **符箓** | 远程输出 / 控制 | 快捷键绘制 + 释放 | 灵力 |
| **古篆文** | 法阵 / 辅助 / 被动 | 镶嵌触发 / 主动激活 | 灵力 + 符文能量 |
| **剑诀** | 近战 / 连招 | 普通攻击 + 技能连段 | 体力 |

## 一、属性体系

### 1.1 基础属性（Primary Attributes）

| 属性 | 缩写 | 说明 | 初始值 | 每级成长 |
|------|------|------|--------|----------|
| **生命** | HP | 归零则战斗失败 | 100 | +15/级 |
| **灵力** | MP | 释放符箓和古篆文消耗 | 50 | +8/级 |
| **体力** | SP | 闪避、近战攻击消耗，自动恢复 | 30 | +4/级 |
| **攻击** | ATK | 影响所有伤害类型的基础值 | 10 | +2/级 |
| **防御** | DEF | 减少受到的伤害 | 5 | +1/级 |
| **速度** | SPD | 移动速度、攻击速度、符箓绘制速度 | 100 | +1/级 |

> 基础属性每升一级自动增长。突破大境界时获得额外属性跃升（见境界系数表）。

### 1.2 二级属性（Secondary Attributes）

二级属性由基础属性 + 装备 + 经脉 + 古篆文共同计算得出。

#### 进攻型

| 属性 | 说明 | 计算公式 |
|------|------|----------|
| **暴击率** | 攻击造成暴击的概率 | `基础 3% + (灵敏×0.05%) + 装备加成 + 经脉加成` |
| **暴击伤害** | 暴击时伤害倍率 | `基础 150% + 装备加成 + 经脉加成` |
| **命中率** | 攻击命中目标的概率 | `基础 95% + (命中等级×2%)` |
| **穿透** | 无视目标防御的比例 | `0% ~ 80%`，由装备和技能提供 |
| **破甲** | 降低目标防御值的比例 | `攻击命中后，目标 DEF × (1 - 破甲%)` |
| **攻击速度** | 普通攻击和技能释放频率 | `基础 1.0 × (1 + SPD/500)` |
| **施法速度** | 符箓绘制和古篆文激活速度 | `基础 1.0 × (1 + SPD/400)` |
| **冷却缩减** | 技能冷却时间减少比例 | `上限 40%`，由装备/经脉/古篆文提供 |

#### 防御型

| 属性 | 说明 | 计算公式 |
|------|------|----------|
| **闪避率** | 完全躲避攻击的概率 | `基础 2% + (灵敏×0.03%)`，上限 30% |
| **格挡率** | 减少受到伤害的概率（减半） | `基础 0%`，由装备和功法提供 |
| **韧性** | 减少被暴击的概率和暴击伤害 | `减少暴击率：韧性×0.5%；暴击伤害 - 韧性×1%` |
| **伤害减免** | 直接减少受到的百分比伤害 | `由防御提供：DEF / (DEF + 500)` |
| **护盾效率** | 护盾吸收伤害的比例 | `基础 100%`，受装备品质影响 |
| **生命恢复** | 每秒恢复生命值 | `基础 1/s + 体质×0.2/s + 装备加成` |
| **灵力恢复** | 每秒恢复灵力值 | `基础 2/s + 神识×0.3/s + 装备加成` |
| **体力恢复** | 每秒恢复体力值 | `基础 3/s + 体质×0.1/s` |

### 1.3 五行深度属性

#### 五行亲和

| 属性 | 说明 | 效果 |
|------|------|------|
| **金亲和** | 金属性伤害加成 | 每点 +2% 金属性伤害 |
| **木亲和** | 木属性伤害加成 | 每点 +2% 木属性伤害 |
| **水亲和** | 水属性伤害加成 | 每点 +2% 水属性伤害 |
| **火亲和** | 火属性伤害加成 | 每点 +2% 火属性伤害 |
| **土亲和** | 土属性伤害加成 | 每点 +2% 土属性伤害 |

> 五行亲和初始值为 0，通过经脉穴位、装备词条、古篆文镶嵌提升。
> 上限：基础 50 点 + 额外加成无上限（衰减曲线：超过 50 点后每点效果减半）。

#### 五行抗性

| 属性 | 说明 | 效果 |
|------|------|------|
| **金抗性** | 减免金属性伤害 | 每点 -1.5% 金属性伤害 |
| **木抗性** | 减免木属性伤害 | 每点 -1.5% 木属性伤害 |
| **水抗性** | 减免水属性伤害 | 每点 -1.5% 水属性伤害 |
| **火抗性** | 减免火属性伤害 | 每点 -1.5% 火属性伤害 |
| **土抗性** | 减免土属性伤害 | 每点 -1.5% 土属性伤害 |

> 五行抗性初始值为 0，上限 75 点（即最多减免 75% ×1.5% = 112.5%，实际取上限 75% 减免）。

#### 五行掌握

| 属性 | 说明 | 效果 |
|------|------|------|
| **金掌握** | 金属性特殊效果触发率 | 破甲概率 + 穿透值 |
| **木掌握** | 木属性特殊效果触发率 | 缠绕/中毒概率 + 持续伤害强度 |
| **水掌握** | 水属性特殊效果触发率 | 减速/冰冻概率 + 持续时间 |
| **火掌握** | 火属性特殊效果触发率 | 灼烧概率 + 灼烧伤害 |
| **土掌握** | 土属性特殊效果触发率 | 眩晕/护盾强度 |

> 每点五行掌握 = +1% 对应特殊效果触发率，初始 0%，上限 60%。

## 二、五行相克与相生

### 2.1 五行相克（战斗核心）

```
金 → 木 → 土 → 水 → 火 → 金
```

- 攻击克制属性 → **伤害 × 1.5**
- 攻击被克属性 → **伤害 × 0.75**
- 同属性 → **伤害 × 1.0**

### 2.2 五行相生（辅助增益）

```
金 → 水 → 木 → 火 → 土 → 金
```

当玩家同时拥有相生两系的符箓/技能时，获得额外效果：

| 相生组合 | 增益效果 |
|----------|----------|
| **金生水** | 金属性技能命中后，下一个水属性技能伤害 +10% |
| **水生木** | 水属性技能命中后，下一个木属性技能持续伤害 +15% |
| **木生火** | 木属性技能命中后，下一个火属性技能暴击率 +10% |
| **火生土** | 火属性技能命中后，下一个土属性技能护盾强度 +20% |
| **土生金** | 土属性技能命中后，下一个金属性技能穿透 +15% |

> 相生增益持续 5 秒，叠加刷新持续时间。

## 三、伤害计算公式

### 3.1 最终伤害公式

```
最终伤害 = 基础伤害 × 五行倍率 × 防御减免 × 境界系数 × 暴击倍率 × 增伤倍率
```

### 3.2 分项说明

#### 基础伤害

```
基础伤害 = 技能基础值 + (ATK × 技能攻击系数)
```

| 变量 | 说明 |
|------|------|
| **技能基础值** | 法术自身的固定伤害数值 |
| **ATK** | 玩家的攻击力 |
| **技能攻击系数** | 不同技能有不同的系数（0.3 ~ 3.0） |

#### 五行倍率

```
五行倍率 = 1.0 + (五行亲和 × 0.02) × 相克修正
```

| 情况 | 相克修正 |
|------|----------|
| 攻击克制属性 | × 1.5 |
| 攻击被克属性 | × 0.75 |
| 同属性 | × 1.0 |
| 无属性 | × 1.0 |

#### 防御减免

```
防御减免 = 1 - (目标DEF / (目标DEF + 500)) × (1 - 穿透%)
```

- 当穿透 = 0% 时，减免基于目标 DEF
- 当穿透 = 100% 时，完全无视防御
- 典型值：DEF=100 时减免 ≈ 16.7%；DEF=500 时减免 = 50%

#### 境界系数

```
境界系数 = 1.0 + (攻击方境界等级 - 防御方境界等级) × 0.1
```

| 境界差 | 系数 | 说明 |
|--------|------|------|
| +3（高三大境界） | 1.3 | 显著压制 |
| +1 | 1.1 | 轻微压制 |
| 0 | 1.0 | 同级公平 |
| -1 | 0.9 | 轻微逆袭 |
| -3（低三大境界） | 0.7 | 极难造成伤害 |

> 境界等级映射：凡人=1, 炼气=2, 筑基=3, 金丹=4, 元婴=5, 化神=6, 悟道=7, 羽化=8, 登仙=9

#### 暴击倍率

```
如果 随机(1~100) ≤ 暴击率:
    暴击倍率 = 暴击伤害% (基础 150%)
否则:
    暴击倍率 = 1.0
```

- 暴击率受目标 **韧性** 减益：`实际暴击率 = 暴击率 - (目标韧性 × 0.5%)`

#### 增伤倍率

```
增伤倍率 = 1.0 + 增伤Buff总和 - 减伤Debuff总和
```

- 上限：增伤倍率 ≤ 3.0（即最多 200% 额外增伤）
- 下限：增伤倍率 ≥ 0.1（即至少保留 10% 伤害）

### 3.3 伤害计算示例

```
玩家：ATK=150, 火亲和=30, 暴击率=25%, 暴击伤害=180%
技能：离火诀, 基础值=80, 攻击系数=1.2
目标：DEF=120, 火抗性=15, 境界=金丹(4)
五行：火→金（目标金属性），克制→×1.5
境界差：金丹(4) vs 金丹(4) → ×1.0

步骤：
1. 基础伤害 = 80 + (150 × 1.2) = 80 + 180 = 260
2. 五行倍率 = 1.0 + (30 × 0.02) × 1.5 = 1.0 + 0.6 × 1.5 = 1.9
3. 防御减免 = 1 - (120 / 620) × (1 - 0) = 1 - 0.194 = 0.806
4. 境界系数 = 1.0
5. 暴击未触发 → 暴击倍率 = 1.0
6. 增伤倍率 = 1.0

最终伤害 = 260 × 1.9 × 0.806 × 1.0 × 1.0 × 1.0 = 398
```

## 四、状态效果

### 4.1 基础状态

| 状态 | 效果 | 持续时间 | 常见来源 |
|------|------|----------|----------|
| **中毒** | 每秒 HP -5% × 叠层（最高 5 层） | 5s | 木系符箓、毒瘴区域 |
| **灼烧** | 每秒 HP -8% + DEF -15% | 4s | 火系符箓 |
| **冰冻** | SPD -50% → 2s 后定身（无法行动） | 3s | 水系符箓 |
| **麻痹** | 每 1.5s 中断行动 0.5s | 4s | 雷系符箓 |
| **缠绕** | 无法移动，闪避率归零 | 3s | 木系符箓、藤蔓法阵 |
| **眩晕** | 无法行动 + 闪避率归零 | 1.5s | 土系符箓、重击 |
| **破甲** | DEF -30% | 6s | 金系符箓、破甲剑诀 |
| **减速** | SPD -40% | 4s | 水系符箓 |
| **沉默** | 无法使用符箓和古篆文 | 3s | 特殊技能 |
| **致盲** | 命中率 -50% | 4s | 暗系古篆文 |

### 4.2 增益状态

| 状态 | 效果 | 持续时间 | 常见来源 |
|------|------|----------|----------|
| **护盾** | 吸收伤害（基于施法者五行亲和） | 8s | 土系符箓、护阵 |
| **灵力充沛** | MP 恢复 +300% | 6s | 聚灵阵、丹药 |
| **金刚** | DEF +50%，免疫负面状态 | 5s | 金刚符 |
| **疾风** | SPD +30%，闪避率 +10% | 6s | 风系古篆文 |
| **嗜血** | 伤害的 15% 转化为 HP 恢复 | 5s | 魔道功法 |
| **霸体** | 免疫控制效果，DEF +20% | 3s | 特定剑诀 |

### 4.3 状态叠加规则

- **同类状态不叠加**（取最高效果），但持续时间独立刷新
- **中毒** 可叠层（最高 5 层），每层独立计时
- 增益状态不可叠加，后释放的覆盖先释放的
- 控制状态优先级：眩晕 > 冰冻 > 麻痹 > 缠绕 > 减速

## 六、属性收益衰减曲线

为防止属性堆叠导致数值爆炸，采用分段衰减机制：

| 属性区间 | 衰减系数 | 示例（以暴击率为例） |
|----------|----------|---------------------|
| 基础区（0~30%） | ×1.0 全额 | 堆到 30% = 实际 30% |
| 溢出区（30%~50%） | ×0.5 半额 | 再堆 20% → 实际 +10%，总计 40% |
| 饱和区（50%~70%） | ×0.25 | 再堆 20% → 实际 +5%，总计 45% |
| 极限区（70%+） | ×0.1 | 再堆 30% → 实际 +3%，总计 48%（上限 50%） |

> 五行亲和也适用此衰减（50 点后半额，100 点后 0.25 倍）。
> 冷却缩减上限 40%，不受衰减影响。
