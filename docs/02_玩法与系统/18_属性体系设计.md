# 属性体系设计（天道九层框架）

> 本章以**天道九层哲学体系**为骨架，重新定义《云汲仙田录》的全部游戏属性。
>
> 天道九层并非九个独立属性系统，而是**同一套属性从精微到显化的九个层次**——
> 一元是本体，二气是动力，三才是结构，四象是框架，五行是材质，
> 六合是空间，七星是节律，八卦是功能，九宫是定数。
>
> 📐 **与现有系统的关系**：本章定义"属性是什么"，[`02_战斗系统.md`](02_玩法与系统/02_战斗系统.md) 定义"属性怎么用于战斗"，
> [`15_法术大全.md`](02_玩法与系统/15_法术大全.md) 定义"法术的具体数值"。
> 三者互补，互不冲突。

## 第一章：一元·混元 — 混沌属性（本体）

> 混元是九层之**体**，对应属性体系的终极维度。登仙前仅有微量体现，登仙后方可真正掌握。

### 1.1 属性表

| 属性 | 缩写 | 说明 | 获取方式 |
|------|------|------|----------|
| **混元力** | PRM | 全属性基础倍率，每点 +0.5% 全属性 | 混沌篆收集、登仙突破、隐藏秘境 |
| **混沌亲和** | CHA | 真实伤害转化率，每点 +1% 伤害转化为真实伤害 | 混沌套装、混元功 |
| **先天一炁** | QI | 突破成功率加成，每点 +2% 大境界突破成功率 | 初始自带（固定值10）、混沌丹药 |

### 1.2 数值范围

| 属性 | 凡人 | 炼气~化神 | 悟道~羽化 | 登仙 |
|------|------|----------|----------|------|
| 混元力 | 0 | 0~5（隐藏） | 5~15 | 15~50 |
| 混沌亲和 | 0 | 0~3 | 3~10 | 10~30 |
| 先天一炁 | 10（固定） | 10~15 | 15~25 | 25~50 |

### 1.3 核心公式

```
真实伤害比例 = min(混沌亲和 × 1%, 80%)
全属性倍率 = 1.0 + 混元力 × 0.005
突破成功率修正 = 基础成功率 × (1 + 先天一炁 × 0.02)
```

## 第二章：二气·阴阳 — 力量与耐力（基础二元）

> 阴阳是九层之**用**，是修士最早接触的属性维度。
> **力量（阳）** 代表进取、攻击、爆发；
> **耐力（阴）** 代表承载、防御、持久。
>
> 🌟 **玩家明确要求：力量与耐力作为独立核心属性。**

### 2.1 属性表

| 属性 | 缩写 | 阴阳 | 说明 | 初始值 | 每级成长 |
|------|------|------|------|--------|----------|
| **力量** | STR | 阳·刚 | 物理攻击系数 + 近战伤害加成 | 8 | +1.5/级 |
| **耐力** | END | 阴·柔 | 生命上限系数 + 伤害减免 | 8 | +1.5/级 |

### 2.2 力量详解

| 项目 | 内容 |
|------|------|
| **定位** | 近战/物理流派的核心属性 |
| **效果 1** | 每点力量 +2 ATK（攻击力） |
| **效果 2** | 每点力量 +0.5% 近战技能伤害（剑诀/体术） |
| **效果 3** | 每点力量 +0.3% 暴击伤害（仅近战） |
| **效果 4** | 力量 ≥ 耐力×2 时触发「刚极」：ATK 再 +10% |
| **来源** | 剑诀功法升级、体修经脉穴位、力量型装备词条 |

```
力量对 ATK 的贡献 = STR × 2
近战伤害加成 = 1.0 + STR × 0.005
```

### 2.3 耐力详解

| 项目 | 内容 |
|------|------|
| **定位** | 生存/坦克流派的核心属性 |
| **效果 1** | 每点耐力 +0.5% HP 上限（对基础 HP 生效） |
| **效果 2** | 每点耐力 +0.2% 伤害减免（独立减伤，与防御乘算） |
| **效果 3** | 每点耐力 +0.3/s 生命恢复 |
| **效果 4** | 耐力 ≥ 力量×2 时触发「柔极」：受到伤害 -10% |
| **来源** | 炼体功法升级、防御型经脉穴位、耐力型装备词条 |

```
HP 上限修正 = 基础HP × (1 + END × 0.005)
耐力减伤 = 1 - (END × 0.002)   // 50 耐力时减伤 10%
```

### 2.4 阴阳平衡机制

力量与耐力之间存在 **阴阳互根** 关系：

| 状态 | 条件 | 效果 |
|------|------|------|
| **阴阳平衡** | `|STR - END| ≤ 5` | 全属性 +3% |
| **阳极** | `STR - END > 5` | 攻击 +5%，受到伤害 +3% |
| **阴极** | `END - STR > 5` | 防御 +5%，伤害输出 -3% |
| **阴阳失调** | `|STR - END| > 20` | 失衡惩罚：灵力消耗 +15% |

> 鼓励玩家根据流派偏向选择力量或耐力，但不宜极端失衡。

## 第三章：三才·天地人 — 三维主属性（结构）

> 三才是九层之**结构**，为修士提供三维成长框架：
> **天·神识**（法术系）、**地·体魄**（防御系）、**人·身法**（技巧系）。

### 3.1 属性表

| 属性 | 缩写 | 三才 | 说明 | 初始值 | 每级成长 |
|------|------|------|------|--------|----------|
| **神识** | INT | 天·法 | 法术攻击系数 + 灵力上限 | 8 | +1.5/级 |
| **体魄** | CON | 地·体 | 防御系数 + 生命恢复 | 8 | +1.2/级 |
| **身法** | DEX | 人·技 | 速度 + 闪避 + 命中 | 8 | +1.2/级 |

### 3.2 神识详解（法术加成）

> 🌟 **玩家明确要求：神识作为法术加成核心属性。**

| 项目 | 内容 |
|------|------|
| **定位** | 法系流派的核心属性 |
| **效果 1** | 每点神识 +1.5 MP（灵力上限） |
| **效果 2** | 每点神识 +0.5% 法术伤害（符箓/古篆文） |
| **效果 3** | 每点神识 +0.4% 法术暴击率 |
| **效果 4** | 每点神识 +0.3/s 灵力恢复 |
| **效果 5** | 神识 ≥ 30 时解锁「神识外放」：范围感知 +1 格 |
| **来源** | 符箓功法升级、古篆文修为、神识型装备词条、头部经脉穴位 |

```
法术伤害加成 = 1.0 + INT × 0.005
灵力上限 = 基础MP + INT × 1.5
法术暴击率 = 基础 3% + INT × 0.004
```

### 3.3 体魄详解

| 项目 | 内容 |
|------|------|
| **定位** | 防御/续航流派的核心属性 |
| **效果 1** | 每点体魄 +1.5 DEF（防御力） |
| **效果 2** | 每点体魄 +0.5/s 生命恢复 |
| **效果 3** | 每点体魄 +1% 治疗效果（受到的治疗量） |
| **效果 4** | 体魄 ≥ 30 时解锁「铜皮铁骨」：受到物理伤害 -5% |
| **来源** | 炼体功法、防御型经脉穴位、体魄型装备词条、躯干经脉穴位 |

```
防御力 = 基础DEF + CON × 1.5
生命恢复 = 基础恢复 + CON × 0.5
```

### 3.4 身法详解

| 项目 | 内容 |
|------|------|
| **定位** | 敏捷/技巧流派的核心属性 |
| **效果 1** | 每点身法 +0.5% 移动速度 |
| **效果 2** | 每点身法 +0.3% 闪避率 |
| **效果 3** | 每点身法 +0.3% 命中率 |
| **效果 4** | 每点身法 +0.2% 攻击速度 |
| **效果 5** | 身法 ≥ 30 时解锁「踏风」：闪避后下次攻击必中 +50% 伤害 |
| **来源** | 轻功功法、身法型经脉穴位、身法型装备词条、足部经脉穴位 |

```
移动速度 = 基础SPD × (1 + DEX × 0.005)
闪避率 = 基础 2% + DEX × 0.003
命中率 = 基础 95% + DEX × 0.003
```

### 3.5 三才流派倾向

| 流派 | 主属性 | 副属性 | 战斗风格 |
|------|--------|--------|----------|
| **符箓天师** | 神识 | 身法 | 远程风筝，法术爆发 |
| **剑阵宗师** | 力量 | 体魄 | 近战硬刚，连招输出 |
| **阵脉宗师** | 神识 | 耐力 | 控制 + 持续伤害，阵法师 |
| **体修金刚** | 力量 | 体魄 | 纯近战坦克，高攻高防 |
| **符剑双修** | 力量+神识均衡 | 身法 | 远近兼顾，操作复杂 |

## 第六章：六合·天地四方 — 空间属性

> 六合是九层之**空间**，标志着属性体系从"元素构成"进入"空间维度"。

### 6.1 属性表

| 属性 | 缩写 | 说明 | 初始值 | 上限 |
|------|------|------|--------|------|
| **技能范围** | RNG | 技能作用范围扩大百分比 | 0% | 50% |
| **无视防御** | PEN | 无视防御比例增加 | 0% | 60%（与装备/技能穿透叠加） |
| **空间掌控** | SPACE | 传送距离 + 闪避位移距离 | 0% | 30% |

### 6.2 技能范围

| 项目 | 内容 |
|------|------|
| **效果** | 所有范围技能的作用半径/宽度增加 |
| **公式** | `实际范围 = 基础范围 × (1 + RNG%)` |
| **来源** | 六合类古篆文、空间属性装备词条、特定阵法加成 |
| **典型值** | 10% → 3格技能变 3.3格；30% → 3格变 3.9格（向上取整） |

### 6.3 无视防御

| 项目 | 内容 |
|------|------|
| **效果** | 与装备/技能提供的穿透叠加，但受上限约束 |

> 九宫是九层之**极**（数理终极），对应游戏中的"运气修正"——不让玩家完全被随机性支配。

### 9.1 属性表

| 属性 | 缩写 | 说明 | 初始值 | 上限 |
|------|------|------|--------|------|

|------|------|

| **机制** | 当连续随机判定失败次数 > 定数阈值时，下次判定必定成功 |
| **公式** | `保底触发数 = max(3, 10 - DEST × 0.1)` |
| **示例** | 定数=50 → 连续 5 次失败后第 6 次必定成功 |
| **来源** | 九宫类古篆文、突破悟道期解锁 |

|------|------|

## 第十章：属性总公式与计算优先级

### 10.1 属性计算层级

所有属性的最终数值按以下优先级计算：

```
第 1 层：基础值（初始值 + 等级成长）
     ↓
第 2 层：装备加成（装备词条 + 套装效果）
     ↓
第 3 层：经脉加成（穴位激活 + 周天循环）
     ↓
第 4 层：古篆文加成（刻印 + 套装效果）
     ↓
第 5 层：功法加成（当前功法 + 功法等级）
     ↓
第 6 层：丹药 Buff（临时 + 永久）
     ↓
第 7 层：七曜/昼夜周期修正（随时间波动）
     ↓
第 8 层：混元力倍率（全属性 × (1 + PRM × 0.005)）
```

### 12.2 战斗系统的伤害公式调整

原有的伤害公式：

```
最终伤害 = 基础伤害 × 五行倍率 × 防御减免 × 境界系数 × 暴击倍率 × 增伤倍率
```

调整后：

```
// 基础伤害分近战/法术
近战基础伤害 = 技能基础值 + ((力量 × 2 + 基础ATK + 装备ATK) × 技能攻击系数) × (1 + 力量 × 0.005)
法术基础伤害 = 法术基础值 + ((神识 × 1.5 + 基础ATK + 装备ATK) × 技能攻击系数) × (1 + 神识 × 0.005)

// 最终伤害
最终伤害 = 基础伤害 × 五行倍率 × 防御减免 × 境界系数 × 暴击倍率 × 增伤倍率 × 混元倍率
```

> 公式中混元倍率默认为 1.0（登仙前），不影响现有数值平衡。
