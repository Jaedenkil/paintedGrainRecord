# 古篆字符文系统

> 本章详细定义古篆文字符的收集、解读、镶嵌与激活机制。

---

## 系统概述

古篆字符文系统是《云汲仙田录》最具特色的核心机制。玩家在探索中 **收集失传的古篆字**，每个字代表天地间的一种 **基础法则**，可镶嵌在装备或法阵中，激活强大的被动或主动效果。

古篆文的设计灵感来源于 **甲骨文** 和 **金文**，赋予游戏浓厚的文化底蕴。

---

## 古篆文分类

### 自然法则类

| 古篆字 | 含义 | 效果 |
|--------|------|------|
| ☰ 天 | 天之力 | 提升所有灵力恢复速度 |
| ☷ 地 | 地之力 | 提升防御和生命恢复 |
| ☵ 水 | 水之力 | 水属性伤害加成 |
| ☲ 火 | 火之力 | 火属性伤害加成 |
| ☳ 雷 | 雷之力 | 雷属性伤害 + 麻痹概率 |
| ☴ 风 | 风之力 | 移动速度 + 攻击速度 |

### 生物灵兽类

| 古篆字 | 含义 | 效果 |
|--------|------|------|
| 龍 | 龙威 | 对妖兽额外伤害 |
| 鳳 | 凤焰 | 火焰范围爆炸 |
| 虎 | 虎魄 | 近战暴击率提升 |
| 龜 | 龟甲 | 受伤后有概率触发护盾 |

### 抽象概念类

| 古篆字 | 含义 | 效果 |
|--------|------|------|
| 生 | 生命 | 击杀回复生命 |
| 死 | 死亡 | 对低生命敌人斩杀 |
| 光 | 光明 | 照亮范围 + 隐匿敌人显形 |
| 暗 | 黑暗 | 减少敌人视野 + 背刺加成 |
| 時 | 时间 | 击杀后技能冷却加速 |
| 空 | 空间 | 闪避距离增加 |

## 收集方式

| 途径 | 说明 |
|------|------|
| 秘境探索 | 在古墓/遗迹中找到石碑，通过解谜获得 |
| 战斗掉落 | 精英/Boss 有概率掉落古篆文碎片 |
| 任务奖励 | 特定支线任务奖励稀有古篆文 |
| 灵田种植 | 极低概率从灵田中挖出古篆文碎片 |
| 合成 | 5 个同品质碎片合成完整古篆文 |

## 镶嵌与激活

### 镶嵌槽位
- 每个装备有 **1~3 个** 古篆文镶嵌槽
- 法阵盘有 **3~6 个** 镶嵌槽

### 套装效果
当装备中同时镶嵌特定组合的古篆文时，激活 **套装共鸣**：

| 组合 | 套装效果 |
|------|----------|
| 天 + 地 + 人 | 三才阵：全属性 +10% |
| 火 + 雷 + 风 | 天罚阵：攻击时触发追加雷击 |
| 龍 + 鳳 + 虎 + 龜 | 四灵阵：全抗性 +20%，四灵护体 |
| 生 + 死 + 時 + 空 | 轮回阵：濒死时满血复活（长冷却） |
