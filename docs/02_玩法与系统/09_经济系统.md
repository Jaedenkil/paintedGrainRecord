# 经济系统

> 本章定义游戏内的货币体系、物价标准、交易规则与经济循环控制。

---

## 一、货币体系

### 1.1 四种货币总览

| 货币 | 说明 | 主要获取 | 主要用途 | 存储上限 |
|------|------|---------|---------|---------|
| **灵石** | 基础货币，修仙界硬通货 | 战斗掉落、任务奖励、灵田产出、商店出售物品 | 日常购买、修理装备、传送费用、炼丹材料 | 999,999 |
| **灵玉** | 高级货币，高纯度灵石精华 | Boss 掉落、特殊事件、拍卖行成交、分解传说装备 | 稀有物品、拍卖行竞价、高级功法、特殊服务 | 9,999 |
| **宗门贡献** | 宗门内流通的贡献点数 | 宗门任务、宗门捐献（灵石→贡献）、宗门活动 | 兑换宗门功法/装备/丹药、宗门升级 | 99,999 |
| **声望** | 与各势力的关系数值 | 完成任务、帮助 NPC、势力专属事件 | 解锁特殊对话/商店折扣/隐藏任务 | 崇拜（≥2000） |

### 1.2 货币兑换规则

| 兑换方向 | 比例 | 限制 | 兑换点 |
|---------|------|------|--------|
| 灵石 → 宗门贡献 | 100:1 | 每日上限 500 贡献 | 宗门执事堂 |
| 灵石 → 灵玉 | ❌ 不可直接兑换 | — | — |
| 灵玉 → 灵石 | 1:100（黑市） | 每日上限 10 次 | 暗影楼黑市（需解锁） |
| 灵石 → 声望 | ❌ 不可直接购买 | — | — |

> **设计意图**：灵玉不可从灵石直接兑换，确保高级货币的稀缺性。玩家必须通过挑战高难度内容（Boss/特殊事件/拍卖）获取灵玉。

### 1.3 灵石购买力参考

| 价格区间 | 可购买内容 | 举例 |
|---------|-----------|------|
| 10~50 | 基础消耗品 | 回灵丹 ×1（30）、清水 ×1（10）、符纸 ×10（20） |
| 50~200 | 常见材料 | 凝露草 ×5（75）、铁矿石 ×5（100）、灵泉水 ×1（60） |
| 200~500 | 中等材料/基础装备 | 良品武器（300~500）、金线莲 ×3（240）、丹方（350） |
| 500~2000 | 稀有材料/功法 | 紫芝 ×1（600）、青铜炉（800）、基础功法（1200） |
| 2000~10000 | 高品质装备/高级材料 | 极品武器（3000~6000）、玄铁炉（5000） |
| 10000+ | 传说装备/房产升级 | 传说武器（15000~30000）、洞府扩建（20000） |

## 三、拍卖行

### 3.1 拍卖流程

```
玩家上架 → 设置起拍价/一口价/持续时间 → 其他玩家竞价 → 结束 → 邮件结算
```

### 3.2 拍卖规则

| 参数 | 值 |
|------|-----|
| 上架费 | 起拍价的 2%（最低 10 灵石，最高 500 灵石） |
| 成交税 | 成交价的 5%（系统回收，用于通胀控制） |
| 持续时间 | 24h / 48h / 72h（玩家选择） |
| 竞价增量 | 起拍价 <1000：+10 / <10000：+50 / <100000：+200 / ≥100000：+1000 |
| 一口价 | 设置起拍价的 2~5 倍，买家可立即成交 |
| 流拍 | 物品通过邮件退回，上架费不退 |
| 竞拍货币 | 仅限灵玉（确保高级货币流通） |
| 上架限制 | 同时最多上架 5 件 |

### 3.3 拍卖行商品类型

| 类型 | 说明 | 常见成交价（灵玉） |
|------|------|-----------------|
| 传说装备 | Boss 掉落/极品锻造 | 5~50 |
| 完整古篆文 | 稀有/高阶篆字 | 3~30 |
| 高级丹方/图纸 | 元婴+ | 2~15 |
| 特殊材料 | Boss 独有掉落 | 1~10 |
| 功法秘籍 | 宗门/隐藏 | 5~40 |

## 四、物价与价值公式

### 4.1 物品基础价值公式

```
基础价值（灵石）= 品级系数 × 境界系数 × 稀有度系数 × 类型基准值
```

| 参数 | 值 |
|------|-----|
| **品级系数** | 凡品 ×1.0 / 良品 ×2.5 / 极品 ×6.0 / 传说 ×15.0 |
| **境界系数** | 炼气 ×1 / 筑基 ×3 / 金丹 ×8 / 元婴 ×20 / 化神 ×50 / 悟道 ×120 / 羽化 ×300 |
| **稀有度系数** | 常见 ×1.0 /  uncommon ×2.0 / 稀有 ×5.0 / 史诗 ×15.0 / 传说 ×50.0 |
| **类型基准值** | 消耗品 10 / 材料 20 / 符箓 30 / 丹药 40 / 装备 100 / 古篆文 200 / 功法 300 |

### 4.2 价值计算示例

```
物品：极品玄铁剑（金丹期、稀有）
计算：品级系数 6.0 × 境界系数 8 × 稀有度系数 5.0 × 类型基准值 100
     = 6.0 × 8 × 5.0 × 100
     = 24,000 灵石（≈ 24 灵玉）
```

### 4.3 灵石→灵玉价值锚定

| 物品 | 灵石价 | 灵玉价 | 锚定依据 |
|------|--------|--------|---------|
| 破境丹 | 不可灵石购买 | 5 灵玉 | 元婴突破刚需，限定灵玉 |
| 传说武器图纸 | 不可灵石购买 | 20 灵玉 | 最高端制造品 |
| 1 灵玉 | — | — | 锚定 ≈ 5000 灵石购买力（黑市兑换 1:100 是折扣价） |

## 五、经济循环与数值推演

### 5.1 玩家日均经济推演（金丹期参考）

| 项目 | 灵石产出/日 | 说明 |
|------|-----------|------|
| 战斗掉落 | 500~1500 | 刷 50~100 只同级怪，含直接掉落+出售战利品 |
| 任务奖励 | 200~800 | 主线/支线/日常混合 |
| 灵田产出 | 100~500 | 出售灵药/作物 |
| 炼丹/炼器出售 | 300~1000 | 自用剩余出售 |
| 材料采集出售 | 100~300 | 杂项材料 |
| **日均总收入** | **1200~4100** | — |

| 项目 | 灵石消耗/日 | 说明 |
|------|-----------|------|
| 丹药消耗 | 200~500 | 回灵丹/培元丹等日常消耗 |
| 装备修理 | 100~400 | 按战斗频率估算 |
| 传送费用 | 50~200 | 跨场景传送 |
| 材料购买 | 200~800 | 补充短缺材料 |
| 炼丹/炼器成本 | 200~600 | 材料成本 |
| **日均总支出** | **750~2500** | — |

| 结论 | 值 |
|------|-----|
| 日均净赚 | 450~1600 灵石 |
| 积累速度 | 约 3~7 天可买一件极品装备 |
| 灵玉获取 | 约 1~3 天/Boss（金丹期） |
| 经济健康度 | ✅ 正循环，但不会过快膨胀 |

### 5.2 境界阶段经济特征

| 境界 | 经济特征 | 主要消耗方向 | 灵玉获取 |
|------|---------|------------|---------|
| 凡人~炼气 | 入不敷出型，靠初始资金和任务支撑 | 基础丹药、符纸 | 无 |
| 筑基 | 收支平衡，灵田开始产出稳定收益 | 装备升级、炼丹材料 | 极少量（特殊事件） |
| 金丹 | 正向循环，有闲钱投资 | 高级材料、拍卖行、宗门捐献 | Boss掉落（每周 1~3） |
| 元婴+ | 富裕阶段，灵石不再是瓶颈 | 灵玉消费、传说装备、洞府升级 | Boss 稳定掉落 |

## 六、维修与强化费用

### 6.1 装备修理

```
修理费用 = 装备基础价值 × 损坏程度（%）× 0.1
```

| 损坏程度 | 说明 | 修理费比例 |
|---------|------|-----------|
| 轻度（<30%） | 正常使用磨损 | 基础价值 × 0~3% |
| 中度（30~60%） | 多次战斗未修 | 基础价值 × 3~6% |
| 重度（60~90%） | 濒危状态 | 基础价值 × 6~9% |
| 破损（>90%） | 继续使用可能彻底损坏 | 基础价值 × 9~10% |

> 修理在任何铁匠铺进行，瞬间完成。装备彻底损坏时保留 50% 材料回收。

### 6.2 装备强化

| 等级 | 成功率 | 费用（灵石） | 属性加成 | 失败后果 |
|------|--------|------------|---------|---------|
| +1 | 100% | 500 | +5% | — |
| +2 | 90% | 1000 | +10% | 降回 +1 |
| +3 | 80% | 2000 | +15% | 降回 +1 |
| +4 | 60% | 4000 | +20% | 降回 +2 |
| +5 | 40% | 8000 | +25% | 降回 +2 |
| +6 | 25% | 15000 | +30% | 降回 +3（可能损坏） |
| +7 | 15% | 30000 | +35% | 降回 +3（可能损坏） |
| +8 | 8% | 50000 | +40% | 可能直接损坏 |
| +9 | 3% | 100000 | +50% | 可能直接损坏 |

> 强化保护符（万象阁：3 灵玉/张）：失败时装备不降级、不损坏，仅消耗保护符。

## 七、通胀控制策略

### 7.1 灵石回收机制

| 回收渠道 | 回收占比 | 说明 |
|---------|---------|------|
| 传送费用 | 5~10% | 跨场景基础消耗，随距离递增 |
| 装备修理 | 10~15% | 战斗越频繁，修理费越多 |
| 商店购买 | 30~40% | 最大回收渠道，玩家消费即回收 |
| 拍卖行费用 | 5~8% | 上架费 + 成交税 |
| 强化费用 | 15~25% | 强化是"灵石黑洞"，高等级强化消耗极大 |
| 宗门捐献 | 5~10% | 灵石→贡献，灵石被系统回收 |
| 死亡惩罚 | 2~5% | 死亡掉落当前携带灵石的 5%（设下限保护） |

### 7.2 灵玉稀缺性保障

| 策略 | 说明 |
|------|------|
| 不可直接兑换 | 灵石→灵玉不可行，保持灵玉稀缺 |
| 有限产出 | 仅 Boss 掉落/特殊事件/拍卖行成交产出 |
| 刚需绑定 | 传说装备、高级古篆文、破境丹 等必须灵玉购买 |
| 消耗出口 | 强化保护符（3 灵玉/张）、稀有图纸（20 灵玉） |
| 回收稳定 | 每次拍卖行成交税回收 5% 灵玉 |

### 7.3 经济健康度监控指标

| 指标 | 健康区间 | 预警值 | 调整手段 |
|------|---------|--------|---------|
| 灵石通胀率 | <5%/月 | >10%/月 | 降低掉落率、增加修理费、推出新消耗品 |
| 灵玉流通量 | 稳定或微增 | >20%/月 | 增加灵玉消耗渠道、降低 Boss 掉落 |
| 日均净赚/消耗比 | 1:0.6~0.8 | <1:0.4 或 >1:1.2 | 调整掉落/价格/修理费 |
| 玩家平均存款 | 不超过该境界一件极品装备价格 | 超过 3 倍 | 推出新消费内容 |
