# 种族设置系统

> 本章设计游戏内与种族/物种模板相关的所有设置项、配置界面与交互逻辑。
>
> **参考游戏**：WoW（种族/职业系统）、BG3（种族子选项+外观）、DOS2（种族标签+对话）、PoE（升华天赋）、老滚5（种族属性预览）、GW2（文化装备）、黑魂（出身礼物）

---

## 一、设计总览

### 1.1 设置层级

```
种族设置
├─ Level 1: 角色创建设置（一次性，新游戏时配置）
├─ Level 2: 快捷键设置（种族主动技能键位）
├─ Level 3: 显示设置（种族相关视觉元素）
├─ Level 4: 界面设置（种族信息布局）
├─ Level 5: 游戏性设置（种族行为选项）
└─ Level 6: 音频设置（种族音效控制）
```

### 1.2 设置存储方式

| 设置类型 | 存储位置 | 生命周期 | 示例 |
|---------|---------|---------|------|
| **角色绑定** | 存档文件（角色数据） | 该角色永久 | 种族选择、外观定制 |
| **账号绑定** | `user://settings.cfg` | 所有角色共享 | 快捷键、显示选项 |
| **会话临时** | 内存变量 | 当前游戏会话 | 自动形态切换开关 |

### 2.2 种族选择面板（参考：WoW + BG3）

| 元素 | 说明 | 实现方式 |
|------|------|---------|
| **种族网格** | 3×3 图标网格，悬停放大并显示简介 | `GridContainer` + 16×16 像素图标 |
| **雷达图** | 实时显示该种族的 5 维属性初始值对比 | `TextureProgressBar` × 5，六边形排列 |
| **种族天赋预览** | 展开式列表，显示该种族 3~4 个天赋 | `Tree` 节点，可折叠/展开 |
| **推荐流派** | 文字提示，可选流派 + 综合评分（⭐） | `Label` + 星级图标 |
| **世界观描述** | 简短背景故事（2~3 句） | `RichTextLabel`，像素古风字体 |
| **出生技能演示** | 小窗口播放该种族出生技能的动画预览 | `AnimatedSprite2D` 嵌入界面 |

**设计参考**：
- WoW：种族选择时左侧大模型展示，右侧属性/技能列表 → 简化为雷达图+文字
- BG3：种族选择时展示种族特性、身高、移速 → 调整为 2D 像素雷达图
- 老滚5：右侧属性条展示种族差异 → 用 0~10 数值 + 进度条

│  神识 INT    3  ███░░░░░░░  [-][●][+]       │
│  体魄 CON    7  ███████░░░  [-][●][+]       │
│  身法 DEX    1  █░░░░░░░░░  [-][●][+]       │
│                                              │

| 规则 | 说明 |
|:----:|------|
| **可调点数** | 5 点自由分配（参考老滚5的"出生星座"额外加点） |

| **实时预览** | 右下角显示调整后的出生技能效果变化 |
| **重置按钮** | 一键恢复种族默认初始分布 |

│  瞳色:  ○ 赤金  ○ 琥珀  ○ 碧蓝              │

| 定制项 | 种族差异 | 技术实现 |
|--------|---------|---------|
| **肤色** | 人族(肤色3种)、龙族(鳞色3种)、凤族(羽色3种)、鬼修(魂色2种) | 精灵调色板，`modulate` 染色 |
| **瞳色** | 所有种族各有 3~4 种 | 独立 `Sprite2D` 图层 |
| **体型** | 龙族体型范围 +0%~+30%，龟族 +0%~+20%，其他 ±10% | `scale` 参数，影响碰撞体积 |
| **种族特征** | 龙族鳞纹、狐族尾数(1~9)、灵族发光强度、鬼修半透明程度 | 附加精灵图层 |
| **初始服饰** | 各种族 2 套初始服装选择 | `preload` 服装精灵 |

## 三、快捷键设置（Key Bindings）

### 3.1 种族专属动作映射

参考 **WoW** 的种族技能自动加入技能栏 + **PoE** 的升华技能槽位设计。

| 动作名 | 说明 | 默认键位 | 手柄映射 | 绑定级别 |
|--------|------|---------|---------|---------|
| `racial_skill_1` | 种族主动天赋 1（如有） | `R` | LB + X | 角色绑定 |
| `racial_skill_2` | 种族主动天赋 2（如有） | `T` | LB + Y | 角色绑定 |
| `form_switch` | 灵族五行形态切换 | `F1` | LB + B | 角色绑定 |
| `form_cycle` | 灵族形态循环（金→木→水→火→土） | `F2` | LB + A | 角色绑定 |
| `race_transform` | 种族临时变身（如化龙诀） | `G` | RB + X | 技能绑定 |
| `toggle_race_aura` | 切换种族光环显示 | `Ctrl+R` | — | 账号绑定 |

### 3.3 种族技能槽位（参考：WoW 动作条）

```
[HUD 底部技能栏 - 种族特有区域]
┌─────────────────────────────────────────────┐
│  [1]  [2]  [3]  [4]  [5]  [6]  ║  [R]  [T] │
│  基础 基础 基础 基础 基础 基础  ║  龙息  龙威 │
│  技能 技能 技能 技能 技能 技能  ║  🐉    👑  │
└─────────────────────────────────────────────┘
         普通技能栏 ← ║ → 种族技能专属槽位
```

| 特性 | 说明 |
|------|------|
| **种族槽位独立** | 种族技能固定放在竖线分隔后的位置，不与普通技能混淆（参考 WoW 职业/种族技能分离） |
| **种族技能图标** | 带种族边框（如龙族技能框为金色龙纹、鬼修为紫色魂火） |
| **冷却显示** | 独立冷却计时显示，支持 `冷却组` 共用冷却 |
| **不可拖拽** | 种族技能固定位置，不可移出技能栏（但可隐藏） |
| **跨角色继承** | 快捷键设置是账号绑定的，换角色后种族技能自动分配到相同键位 |

### 6.2 灵族自动形态切换规则

参考 **GW2** 的元素使元素切换 + **PoE** 的自动触发机制。

| 规则模式 | 行为 | 适用场景 |
|---------|------|---------|
| **手动切换**（默认） | 玩家按 `F1/F2` 手动切换 | 追求精确操作的玩家 |
| **根据敌人五行** | 自动检测敌人五行属性，切至克制的形态 | 新手/懒人模式 |
| **根据场景五行** | 自动根据当前场景灵气浓度切至对应形态 | 在特定地图自动适应 |
| **循环轮流** | 每 15s 自动切换至下一个形态 | 追求全面覆盖 |
| **智能模式** | 战斗中根据敌人切形态，跑图时根据场景切 | 推荐给中阶玩家 |
