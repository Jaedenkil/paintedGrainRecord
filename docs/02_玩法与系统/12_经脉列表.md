## 二、经脉流注与交接

古中医认为，全身气血在十二正经中 **首尾相衔、如环无端**，按固定顺序周流不息。每条经脉的末端与下一条经脉的起始通过 **络穴、经筋或脏腑** 相互衔接，形成完整的 **十二经脉流注循环**。

打通经脉流注路径上的 **交接节点**，可激活 **流注连锁奖励**。

### 流注循环顺序

以下为十二正经的完整流注顺序（箭头表示气血流向，括号内为交接位置）：

```
手太阴肺经 ──→ 手阳明大肠经 ──→ 足阳明胃经 ──→ 足太阴脾经
    (食指端)          (鼻旁)            (足大趾端)
       ↑                                               ↓
   足厥阴肝经                                   手少阴心经
    (足大趾端)                                      (小指端)
       ↑                                               ↓
   足少阳胆经                                   手太阳小肠经
    (目外眦)                                        (目内眦)
       ↑                                               ↓
   手少阳三焦经 ←── 手厥阴心包经 ←── 足少阴肾经 ←── 足太阳膀胱经
     (无名指端)        (胸中)            (足小趾端)
```

> **流注方向规律**：手三阴经（胸→手）→ 手三阳经（手→头）→ 足三阳经（头→足）→ 足三阴经（足→胸）→ 回到手三阴经，形成一个封闭循环。

### 周天循环体系

周天循环是经脉流注的高级整合机制，分为 **小周天** 与 **大周天** 两个层次。

#### 小周天（任督循环）

小周天指气血沿 **任脉**（前）与 **督脉**（后）循环运转，是筑基期修士打通的基础循环。

```
督脉（后升）：长强 → 命门 → 至阳 → 灵台 → 大椎 → 哑门 → 风府 → 百会
                                                         ↓
任脉（前降）：承浆 ← 天突 ← 膻中 ← 中脘 ← 神阙 ← 气海 ← 关元 ← 会阴
```

**小周天开启条件**：
- 任脉 ≥4 穴激活 + 督脉 ≥4 穴激活
- 境界 ≥ 筑基期

**小周天运转奖励**：

| 层数 | 条件 | 奖励 |
|------|------|------|
| 初通 | 任督各 ≥4 穴 | 生命恢复 +3/s，灵力恢复 +3/s，全属性 +5 |
| 贯通 | 任督各 ≥6 穴 | 额外：受到伤害 -8%，移动速度 +5% |
| 大通 | 任脉 8 穴满 + 督脉 10 穴满 | 额外：解锁被动 **「小周天护体」**—— 受到致命伤害时触发护盾（3 秒无敌，每 120 秒一次） |

#### 大周天（十二经循环）

大周天指气血贯通 **全身十二正经 + 奇经八脉** 的完整循环，金丹期以上方可开启。

**大周天开启条件**：
- 十二正经均已打通（每条至少 1 穴）
- 境界 ≥ 金丹期

**大周天运转奖励**：

| 层数 | 条件 | 奖励 |
|------|------|------|
| 初通 | 十二正经全通 + 任督 ≥4 穴 | 全属性 +10，五行亲和 +5（全） |
| 贯通 | 十二正经每条 ≥3 穴 + 任督各 ≥6 穴 | 额外：所有伤害 +15%，所有减伤 +10% |
| 大成 | 十二正经每条 ≥5 穴 + 任督全满 | 额外：解锁被动 **「大周天归元」**—— 战斗中使用灵力时 20% 概率返还 |
| 圆满 | 全身至少 80 穴激活 + 所有经脉开通 | 额外：解锁终极被动 **「天人合一」**—— 全属性 +25，无视五行克制减免 50% |
