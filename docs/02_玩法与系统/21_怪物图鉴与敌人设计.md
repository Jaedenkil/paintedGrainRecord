# 怪物图鉴与敌人设计

> 本章定义游戏中所有敌人（妖兽、修士、魔物、Boss）的数值模板、行为模式和掉落体系。所有怪物设计均基于 [`18_属性体系设计.md`](02_玩法与系统/18_属性体系设计.md) 的九层天道属性框架和 [`02_战斗系统.md`](02_玩法与系统/02_战斗系统.md) 的伤害公式。

---

## 一、设计总则

### 1.1 敌人分类

| 类别 | 说明 | 智能等级 | 典型行为 |
|:----:|------|:--------:|----------|
| **妖兽** | 自然生灵，受灵气影响变异 | 低～中 | 巡逻→警戒→追击→战斗 |
| **人类修士** | 敌对宗门/散修 | 中～高 | 巡逻→发现→战术战斗→逃跑（HP<20%）|
| **魔物** | 被暗影楼/末法之气污染的怪物 | 低～中 | 巡逻→狂暴冲锋→死战不退 |
| **古篆守卫** | 秘境/遗迹中的机关守卫 | 无（固定AI）| 区域巡逻→锁定入侵者→固定技能循环 |
| **Boss** | 区域统治者/剧情关键敌人 | 高 | 多阶段战斗，含转场机制 |

### 1.2 难度曲线设计原则

1. **区域线性递进**：青竹谷→落霞山脉→幽暗密林→碧波湖泽→炎火深渊→万雷天谷，怪物强度逐区递增
2. **境界对应**：区域怪物境界 = 该区域推荐玩家境界 ±1（低阶区域 +0～+1，高阶区域 -1～+0）
3. **精英稀有度**：每个区域 5% 概率出现精英怪（属性 ×1.5，掉落 ×2），1% 概率出现稀有怪（属性 ×2，专属掉落）
4. **妖兽潮动态难度**：根据玩家等级和区域探索度动态调整怪物密度和属性

### 1.3 怪物通用属性结构

```
{
  "名称": "string",
  "境界": "炼气/筑基/金丹/元婴/化神/悟道",
  "类别": "妖兽/修士/魔物/守卫/Boss",
  "五行属性": ["金","木","水","火","土"]（0～2种）,
  "基础属性": {
    "HP": 数值,
    "攻击": 数值,
    "防御": 数值,
    "速度": 数值
  },
  "五行亲和": { 金:0～100, 木:0～100, 水:0～100, 火:0～100, 土:0～100 },
  "经验值": 数值,
  "技能列表": ["技能1","技能2",...],
  "掉落池": [{ "物品": "id", "概率": 0.xx, "数量": [min,max] }]
}
```

### 3.2 最终 Boss：暗影楼主

> **触发条件**：完成主线第六章「末法之源」后，在万雷天谷最深处进入"暗影之殿"。

**基本信息**：
- 境界：半步化神（压制玩家一个大境界）
- 五行为暗（无克制关系，所有五行伤害 ×0.7）
- HP：12000 | 攻击：350 | 防御：120 | 速度：30

**四阶段战斗**：

| 阶段 | HP 区间 | 行为 | 特殊机制 |
|:----:|:-------:|------|----------|
| **P1·暗影帷幕** | 100%～70% | 暗影弹幕 + 瞬移 | 场景昏暗，玩家视野缩小 30% |
| **P2·末法之蚀** | 70%～40% | 召唤暗影分身 ×2 + 全屏AOE | 分身攻击附带末法印记（3层即死）|
| **P3·真相揭露** | 40%～15% | 揭面→真身（攻击力+50%）| 剧情对话插入，Boss 展示真实身份 |
| **P4·同归于尽** | 15%～0% | 自爆倒计时 60s | 场景崩塌，DPS 检测，失败即全灭 |

**掉落**：
- 暗影令 ×1（100%，触发隐藏剧情）
- 末法核心 ×1（70%，传说炼器材料）
- 暗影楼密卷 ×1（50%，解锁暗影楼势力线）
- 羽化丹丹方 ×1（10%，化神突破所需）

### 3.3 隐藏 Boss：天道化身

> **触发条件**：完成隐藏任务链「九层天道」+ 收集全部 108 古篆文 + 灵田等级 10。

**基本信息**：
- 境界：悟道（超越玩家 1～2 大境界）
- 五行为全（五行抗性均为 80）
- HP：30000 | 攻击：500 | 防御：200 | 速度：40

**特殊机制**：
- 每 10s 切换天道属性（一元→二气→…→九宫），对应属性抗性 +90%，其他抗性 -30%
- 需要玩家使用对应天道属性的技能/符箓攻击才能造成有效伤害
- 击败后触发结局 C「天道合一」必要条件之一

**掉落**：
- 天道碎片 ×1（100%，合成天道至宝）
- 混元珠 ×1（50%，全属性 +20）
- 称号「逆天者」（唯一）

### 5.2 精英怪规则

- 每个区域固定 1～2 个精英刷新点
- 精英刷新概率：基础 5%，每击败 20 只普通怪增加 5%（上限 30%）
- 精英怪掉落池含专属物品
- 精英怪标记（金色边框）在 2.5D 方块场景中显眼可见

### 5.3 妖兽潮事件

> 随机事件，触发时该区域怪物密度 ×3，持续 10 分钟。

**触发条件**：
- 基础概率：每游戏日 5%
- 玩家击杀数超过区域阈值（50 只/日）
- 天道参数·七曜特定组合
- 可被 [`14_篆路自动化系统.md`](02_玩法与系统/14_篆路自动化系统.md) 的警戒篆路提前预警

**妖兽潮奖励**（击退后）：
- 灵石 ×（区域基准 × 5）
- 区域专属材料箱 ×1（含 3～5 种随机材料）
- 区域声望 +50
