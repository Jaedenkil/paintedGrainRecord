# 成就系统

> 本章定义游戏的成就体系，包括成就分类、代表性示例、奖励机制和 UI 交互。成就系统从 [`08_任务与事件.md`](02_玩法与系统/08_任务与事件.md) 的 8.3 节中抽取并独立成章。以下列出各类型的基础示例，完整成就列表在开发阶段通过 `AchievementData.tres` 资源文件配置。

---

## 一、成就分类

### 1.1 六大类别

| 类别 | 侧重方向 | 示例成就数 | 总成就点数（示例）|
|:----:|:---------|:----------:|:----------------:|
| **修炼** | 境界突破、技能树、经脉 | 3 | 65 |
| **战斗** | 击败怪物/Boss、连击、无伤 | 3 | 80 |
| **经营** | 灵田种植、炼丹炼器、经济 | 3 | 60 |
| **收集** | 古篆文、装备、灵植图鉴 | 2 | 45 |
| **探索** | 场景发现、隐藏任务、事件 | 2 | 50 |
| **隐藏** | 特殊条件、彩蛋、终极目标 | 2 | 200 |
| **总计** | — | **15** | **500** |

### 1.2 成就稀有度

| 稀有度 | 达成率预期 | 成就点数 | 标识色 |
|:------:|:---------:|:--------:|:------:|
| **普通** | >50% 玩家 | 10～20 | 铜色 |
| **稀有** | 15%～50% 玩家 | 25～40 | 银色 |
| **史诗** | 3%～15% 玩家 | 50～75 | 金色 |
| **传说** | <3% 玩家 | 100 | 彩虹/动态 |

## 三、成就奖励体系

### 3.1 称号系统

称号是成就系统的主要奖励之一，装备后提供属性加成。玩家同时只能激活一个称号。

| 称号 | 获取方式 | 属性加成 | 解锁条件 |
|:----:|:---------|:--------:|:---------|
| **化神尊者** | 成就#3 | 全属性 +10 | 化神突破 |
| **逆天者** | 成就#6 | 天道亲和 +20 | 击败天道化身 |
| **万象灵田** | 成就#8 | 灵田产量 +20% | 灵田 Lv.10 |
| **富甲一方** | 成就#9 | 交易价格 -5% | 累计 100000 灵石 |
| **万象归篆** | 成就#11 | 全属性 +15 | 集齐 108 篆 |
| **天道行者** | 成就#13 | 天道参数 +20% | 九层极端值 |
| **天道合一** | 成就#15 | 解锁飞升后内容 | 结局 C |

### 3.2 成就点数里程碑

累计成就点数达到特定阈值时，解锁全局奖励：

| 成就点数 | 奖励 |
|:--------:|------|
| 100 | 背包扩容 +10 格 |
| 250 | 称号槽位 +1（可同时装备 2 个称号）|
| 500 | 解锁隐藏场景「天道碑林」|

### 3.3 奖励发放机制

1. **即时通知**：达成成就时，屏幕右上角弹出成就通知（含成就图标 + 名称 + 点数）
2. **奖励自动发放**：灵石/物品直接进入背包；称号自动解锁
3. **首次领取动画**：首次获得成就时播放约 2s 的庆祝动画
4. **成就跳转**：点击通知可直接跳转到成就面板查看详情

## 四、UI 与交互

### 4.1 成就面板布局

```
┌────────────────────────────────────────────────┐
│  🏆 成就系统                       总计: 450/500 │
├──────┬─────────────────────────────────────────┤
│ 全部 │ [修炼] [战斗] [经营] [收集] [探索] [隐藏]  │
├──────┴─────────────────────────────────────────┤
│                                                 │
│  ┌─── 初入仙途 ──────────────────────── [✓]  ┐  │
│  │ 完成序章                                    │  │
│  │  普通 · 10 pts    奖励: 灵石 ×50            │  │
│  └─────────────────────────────────────────────┘  │
│                                                 │
│  ┌─── 金丹大道 ──────────────────────── [ ]  ┐  │
│  │ 突破至金丹期                     进度: 筑基3层│
│  │  稀有 · 30 pts    奖励: 中品灵石 ×5           │  │
│  └─────────────────────────────────────────────┘  │
│                                                 │
├─────────────────────────────────────────────────┤
│  筛选: 全部 | 未完成 | 可领取                     │
└─────────────────────────────────────────────────┘
```

### 4.2 成就通知样式

**获得成就时弹出**（屏幕右上角，持续 4s 后自动消失）：

```
┌──────────────────────────┐
│  🏆 成就解锁！            │
│  ┌──────────────────┐    │
│  │ [Icon] 金丹大道   │    │
│  │ 成就点数 +30      │    │
│  │ 奖励: 中品灵石 ×5 │    │
│  └──────────────────┘    │
│        [查看详情]         │
└──────────────────────────┘
```

### 4.3 进度追踪

- 每个成就下方显示进度条（如有进度概念，如 "击败怪物: 47/100"）
- 可右键成就选择「追踪此成就」，在 HUD 侧边栏显示进度
- 同时最多追踪 3 个成就
- 隐藏类成就在达成前显示「???」和隐藏条件

### 4.4 存档与跨周目继承

- 成就进度保存在 `user://achievements.cfg`（独立于存档）
- 成就跨周目继承（新开存档不重置成就）
- 称号和成就点数奖励在所有存档中共享
