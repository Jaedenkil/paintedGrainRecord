## 一、装备分类

### 1.1 十二槽位体系

| # | 槽位 | 核心属性倾向 | 装备限制 |
|:-:|:----:|:-----------:|:--------:|
| 1 | **武器** | 攻击力、五行伤害 | 流派限制（剑诀/符箓/古篆）|
| 2 | **头** | 防御力、神识 | 无 |
| 3 | **肩** | 防御力、HP | 无 |
| 4 | **颈** | 抗性、恢复、暴击伤害 | 无 |
| 5 | **胸** | 防御力、HP（主防部位） | 无 |
| 6 | **背** | 速度、闪避 | 无 |
| 7 | **腕** | 攻击力、攻击速度 | 无 |
| 8 | **手** | 攻击力、精准 | 无 |
| 9 | **裤** | 防御力、HP | 无 |
| 10 | **鞋** | 速度、闪避 | 无 |
| 11 | **戒指×2** | 暴击、五行伤害 | 限装备2枚，不可同名 |
| 12 | **饰品×2** | 抗性、恢复、特殊效果 | 限装备2件，不可同名 |

**防具部位基础防御系数**（以胸部为基准 1.0）：
| 部位 | 胸 | 头 | 肩 | 裤 | 腕 | 手 | 鞋 |
|:----:|:--:|:--:|:--:|:--:|:--:|:--:|:--:|
| 系数 | 1.0 | 0.7 | 0.6 | 0.6 | 0.4 | 0.3 | 0.3 |

**武器子类型**：剑（均衡）、拂尘（法术）、符笔（五行）、飞剑（远程物理）

## 二、词缀系统

### 2.1 词缀类型

词缀分为 **前缀**（影响基础属性）和 **后缀**（影响特殊效果）。

| 词缀名 | 类型 | 效果（品质等级 Lv.1/2/3） | 出现位置 |
|:------:|:----:|:------------------------:|:--------:|
| **锋锐** | 前缀 | 攻击力 +5/12/25 | 武器、腕、手 |
| **破甲** | 前缀 | 无视防御 +3%/7%/15% | 武器 |

|:------:|:----:|:---------------:|:--------:|

| **福缘** | 后缀 | 掉落品质 +5%/10%/20% | 饰品 |

| **空间** | 后缀 | 技能范围 +5%/10%/20% | 武器、饰品 |
| **神识** | 前缀 | 神识 +3/8/18 | 头、颈 |

### 2.2 词缀生成规则

```
词缀数量 = 品质决定（凡品 0 / 良品 1 / 极品 2 / 传说 3）
词缀品质 = 权重随机（Lv.1 60% / Lv.2 30% / Lv.3 10%）
词缀池选择 = 根据装备类型筛选可用词缀列表
重复规则 = 同一装备不可出现同名词缀
```

**天道参数化修正**（引用 [`17_天道参数化系统.md`](02_玩法与系统/17_天道参数化系统.md)）：
- 锻造时场景的天道指纹影响词缀池分布
- 幽暗密林锻造的武器更易出现木系/毒系词缀
- 炎火深渊锻造的武器更易出现火系/攻击类词缀

引用 [`09_经济系统.md`](02_玩法与系统/09_经济系统.md) 的强化数值：

| 强化等级 | 成功概率 | 消耗灵石 | 辅助材料 | 失败后果 |

| +3 | 80% | 1200 | 基础灵材 ×1 | 降回 +1 |

| +9 | 10% | 50000 | 极品灵石 ×2 + 雷劫石 ×3 | 降回 +4 |
| +10 | 5% | 100000 | 传说灵材 ×1 + 雷劫石 ×5 | 装备破碎 |

**强化属性增长**：每级 +10% 基础属性（乘法叠加）
```
+5 强化：基础属性 × (1.10)^5 = ×1.61
+10 强化：基础属性 × (1.10)^10 = ×2.59
```

## 六、获取途径总览

| 获取方式 | 品质范围 | 说明 |
|:--------:|:--------:|------|
| **怪物掉落** | 凡品～极品 | 普通怪出凡品，精英出良品～极品，Boss 出极品～传说 |
| **锻造（05）** | 凡品～传说 | 品质取决于锻造台等级、材料品质和锤炼小游戏表现 |
| **任务奖励** | 良品～极品 | 特定主线/支线任务的固定奖励 |
| **商铺购买** | 凡品～极品 | 杂货铺出凡品，万象阁出极品，宗门宝库出专有装备 |
| **秘境宝箱** | 极品～传说 | 隐藏秘境中的特殊宝箱，需特定古篆文解锁 |
| **合成** | 传说 | 天道碎片/特定材料组合合成唯一装备 |
