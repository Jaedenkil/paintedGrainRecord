# 生活属性体系

> 本章定义《云汲仙田录》中战斗之外的 **生活职业与生产属性** 体系。
> 生活系统提供资源获取、装备制造、战斗辅助等全方位支持。

---

## 一、生活属性总览

### 1.1 六项核心生活属性

| 属性 | 缩写 | 说明 | 影响范围 |
|------|------|------|----------|
| **采集力** | GTH | 采集资源的速度与品质 | 采矿、采药、伐木、钓鱼 |
| **耕作力** | FARM | 灵田种植的效率与产出 | 播种、生长、收获 |
| **炼造力** | CRAFT | 制造物品的品级与特效 | 炼丹、炼器、烹饪、符箓绘制 |
| **探索力** | EXPL | 地图探索的收益与发现 | 秘境感知、宝箱发现、隐藏通道 |
| **感应力** | SENSE | 对环境和灵气的感知能力 | 灵脉定位、古篆文识别、矿脉探测 |
| **商贾力** | MERC | 交易相关的效率 | 买卖价格、声望获取、任务报酬 |

> 六项生活属性初始值为 **1**，上限 **100**。通过生活活动获取经验逐步提升。

### 1.2 生活属性与战斗属性的关系

```
生活属性 ↔ 战斗属性 相互影响
```

| 生活属性 | 关联战斗属性 | 影响方式 |
|----------|-------------|----------|
| 采集力 (GTH) | 攻击 (ATK)、体力上限 | GTH 每 10 点 → ATK +1；GTH 50 点解锁「破岩击」特技 |
| 耕作力 (FARM) | 生命上限 (HP) | FARM 每 10 点 → HP +10；FARM 60 点解锁「大地亲和」被动 |
| 炼造力 (CRAFT) | 五行亲和 | CRAFT 每 10 点 → 五行亲和 +1；CRAFT 50 点解锁「器灵共鸣」 |
| 探索力 (EXPL) | 速度 (SPD)、闪避率 | EXPL 每 10 点 → SPD +1；EXPL 50 点解锁「险境直觉」 |
| 感应力 (SENSE) | 暴击率、灵力上限 | SENSE 每 10 点 → MP +5；SENSE 50 点解锁「灵气感知」 |
| 商贾力 (MERC) | 灵力恢复 | MERC 每 10 点 → MP 恢复 +0.2/s；MERC 50 点解锁「以物易物」 |

## 二、采集系统

### 2.1 采集属性公式

```
采集产出数量 = 基础数量 × (1 + GTH × 0.01) × 工具倍率 × 品质倍率
采集速度 = 基础时间 / (1 + GTH × 0.005)
稀有产出概率 = 基础概率 × (1 + GTH × 0.02) + SENSE × 0.001
```

## 三、钓鱼系统

### 3.1 钓鱼基础

钓鱼需要 **鱼竿**（工具）和 **鱼饵**（消耗品），在水域场景进行。

| 鱼竿等级 | 解锁境界 | 基础钓鱼时间 | 特殊效果 |
|----------|----------|-------------|----------|
| 竹竿 | 初始 | 10s | 无 |
| 灵竹竿 | 炼气 | 8s | 钓力 +10% |
| 玄铁竿 | 筑基 | 7s | 钓力 +20%，耐久翻倍 |
| 金丝竿 | 金丹 | 6s | 钓力 +35%，可钓稀有鱼 |
| 龙筋竿 | 元婴 | 5s | 钓力 +50%，缩短咬钩时间 |
| 仙骨竿 | 化神 | 4s | 钓力 +80%，概率钓上宝物 |

### 3.2 钓鱼流程

```
选择钓点[1] → 抛竿 → 等待咬钩[2] → 拉锯小游戏[3] → 收获
```

1. **钓点选择**：不同的水域产出不同鱼种。特殊钓点（灵泉、深渊）产出稀有鱼。
2. **咬钩时间**：`基础时间 × (2 - 鱼竿品质×0.2) / (1 + GTH×0.01)`，范围 3~20s
3. **拉锯小游戏**：进度条保持在一定区间，持续 5s 成功。难度受鱼种稀有度影响。

### 4.2 烹饪公式

```
成品品质概率:
  - 传说（金色）：基础 0% + (CRAFT-50)×0.2% + 厨具加成（最高 8%）
  - 完美（紫色）：基础 5% + (CRAFT×0.3)% + 厨具加成（最高 40%）
  - 良好（蓝色）：基础 30% + (CRAFT×0.2)% 
  - 普通（白色）：剩余概率
  - 粗糙（灰色）：基础 20% - (CRAFT×0.15)%（最低 2%）

成品效果倍率: 粗糙×0.5 / 普通×1.0 / 良好×1.3 / 完美×1.8 / 传说×3.0
```

| **鱼塘** | 石制 | 灵石 ×30 + 清水 ×100 | 养殖鱼类（自动繁殖） |

### 5.3 建筑公式

```
建造时间（秒）= 基础时间 / (1 + GTH × 0.01 + CRAFT × 0.005)
建筑耐久度 = 基础耐久 × (1 + GTH × 0.02 + 材料倍率)
建筑效果倍率 = 基础倍率 × (1 + 建筑等级 × 0.2 + 洞府灵气浓度 × 0.1)
```

### 5.4 洞府灵气浓度

洞府灵气浓度影响所有建筑的效率：

| 灵气等级 | 名称 | 解锁方式 | 效果倍率 |
|----------|------|----------|----------|
| Lv.0 | 凡地 | 初始 | ×1.0 |
| Lv.1 | 灵地 | 建造聚灵阵基 | ×1.3 |
| Lv.2 | 宝地 | 灵脉接入 | ×1.6 |
| Lv.3 | 福地 | 灵脉接入 + 聚灵阵 Lv.3 | ×2.0 |
| Lv.4 | 洞天 | 灵脉接入 + 聚灵阵 Lv.5 + 时间阵 | ×2.5 |
| Lv.5 | 仙境 | 特殊剧情解锁 | ×3.0 |
