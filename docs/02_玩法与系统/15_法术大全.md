# 法术大全

> 本章收录《云汲仙田录》中所有可学法术的详细属性数值。
> 法术分为三大体系：**符箓**（远程）、**古篆文**（法则）、**剑诀**（近战）。

---

## 法术通用属性说明

每个法术包含以下属性：

| 属性 | 说明 |
|------|------|
| **法术名称** | 法术的正式名称 |
| **所属系** | 符箓/古篆文/剑诀 |
| **五行属性** | 金/木/水/火/土/无/雷/风/冰（衍生属性） |
| **法术阶位** | 基础/中级/高级/仙级（对应四阶品质） |
| **解锁境界** | 学习该法术所需的最低境界 |
| **基础伤害** | 法术的固定伤害基础值 |
| **攻击系数** | ATK 乘以该系数加入伤害（倍率） |
| **灵力/体力消耗** | 释放法术消耗的资源 |
| **冷却时间** | 释放后需等待的时间（秒） |
| **施法时间** | 从开始到释放的吟唱/前摇时间（秒） |
| **射程/范围** | 法术的有效距离或作用范围（格/tile） |
| **特殊效果** | 附加的状态效果或特殊机制 |

### 伤害计算参考

```
法术伤害 = 法术基础值 + (ATK × 攻击系数) × 五行亲和加成
```

> 完整伤害计算参见 [`02_战斗系统.md`](02_玩法与系统/02_战斗系统.md#三伤害计算公式)。

## 四、法术进阶规则

### 4.1 法术熟练度

每次使用法术增加熟练度，提升后解锁强化效果：

| 熟练等级 | 所需使用次数 | 效果 |
|----------|-------------|------|
| Lv.1 入门 | 初始 | 基础效果 |
| Lv.2 熟练 | 20 次 | 灵力/体力消耗 -10% |
| Lv.3 精通 | 50 次 | 伤害/效果 +10% |
| Lv.4 大师 | 100 次 | 冷却 -15% |
| Lv.5 宗师 | 200 次 | 解锁特殊强化分支（二选一） |

### 4.2 法术强化分支（宗师级）

法术达到 Lv.5 宗师时，可选择以下强化方向之一：

| 法术 | 强化分支 A | 强化分支 B |
|------|-----------|-----------|
| **火球符** | 炎爆：伤害 +40%，范围 +50% | 连珠：消耗减半，连发两发 |
| **冰晶术** | 永冻：冰冻时间 +2s | 碎片：碎裂伤害 +100% |
| **拔刀斩** | 居合：蓄力时间 -0.3s | 一闪：暴击率 +25% |
| **破军** | 破军·势：每段递增伤害 +10% | 破军·破：最终段破甲 100% |

### 4.3 法术切换与装备

- 玩家可同时装备 **6 个符箓** + **3 个剑诀连段** + **2 个古篆文主动法术**
- 战斗中通过快捷键（1~6 / Q/E / Shift+数字）切换
- 符箓在休息区可自由更换（非战斗状态）
- 剑诀连段在篝火/存档点更换
- 古篆文主动法术绑定在经脉功法槽位
