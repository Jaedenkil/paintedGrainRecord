# 任务与事件

> 本章定义任务系统与随机事件机制，包含主线/支线/日常/隐藏任务的具体内容设计与事件系统。

---

## 一、任务类型

### 1.1 任务分类表

| 类型 | 说明 | 数量 | 奖励倾向 | 标识色 |
|------|------|------|---------|--------|
| **主线** | 推动剧情发展，解锁新区域/系统 | ~30 个 | 大量经验、稀有装备、系统解锁 | 金色 |
| **支线** | 世界各地的独立故事 | ~20 个 | 符箓/古篆文/材料/特殊道具 | 蓝色 |
| **日常** | 每日刷新，可重复完成 | 每日 5/10 池 | 灵石、基础材料 | 绿色 |
| **宗门** | 加入宗门后的贡献任务 | 无限刷新 | 宗门贡献、专属功法 | 紫色 |
| **隐藏** | 特定条件触发，无标记 | ~8 个 | 独特奖励、成就、隐藏结局线索 | 无标识 |
| **成就** | 长期目标型，一次性 | ~30 项 | 称号、特殊外观、永久属性 | 红色 |
| **挑战** | 高难度限时任务 | ~5 个 | 传说装备、稀有材料 | 橙色 |

### 1.2 任务活跃规则

| 规则 | 值 |
|------|-----|
| 活跃任务上限 | 10 个 |
| 已完成任务保留 | 可在「任务日志」中回顾 |
| 任务日志快捷键 | `J`（可在设置中自定义） |
| 放弃任务 | 支线/日常可放弃，主线不可 |
| 任务追踪 | HUD 最多追踪 3 个，点击任务日志中的「追踪」设置 |

## 六、任务链规则

### 6.1 链式任务结构

```
任务 A → 任务 B → 任务 C（→ 分支 D1 / D2）
```

| 规则 | 说明 |
|------|------|
| 自动接取 | 前置任务完成后自动接取下一环 |
| 断链处理 | 放弃当前环节 = 整条链重置（从 A 开始） |
| 分支选择 | 部分任务链在特定环节提供 2 选 1 分支，影响后续奖励 |
| 链长度 | 3~5 个环节 |
| 链总奖励 | 最后一环奖励 = 前序累计 ×1.5 倍 |

### 6.2 任务链示例：「暗影之谜」

```
① 暗流涌动（主线） → 发现暗影楼踪迹
       ↓
② 暗影谍影（支线） → 潜入据点获取情报
       ↓
③ 抉择：交给天剑宗 / 交给万象阁 / 自己调查
  ┌────┼────┐
  ↓    ↓    ↓
④天剑宗路线:   ④万象阁路线:     ④独自行路线:
  围剿暗影楼     商业制裁暗影楼    暗影楼地下遗迹
       ↓              ↓               ↓
⑤天剑宗声望+50  ⑤万象阁声望+50   ⑤获得暗影楼传承
  奖励：天剑套装   奖励：万象宝珠    奖励：暗影步法
```

### 7.2 事件交互方式

| 规则 | 说明 |
|------|------|
| 触发提示 | 屏幕中央弹出事件标题 + 选项窗口（2~3 个选项） |
| 时效性 | 秘境之门/流星坠落 等事件有 30s~5min 的响应窗口，错过消失 |
| 声望影响 | 部分选择影响势力声望（+5~+20 / -5~-20） |
| 重复触发 | 灵脉爆发/游方道人 等可重复触发，秘境之门/天降异象 仅一次 |
| 事件冷却 | 同类型事件触发后有 24h（游戏内）冷却 |

### 7.3 事件选项分支示例

```
事件：游方道人
  ├─ 交易 → 打开游方商店（随机 3~5 件商品，价格优惠）
  ├─ 接受考验 → 与道人召唤的幻影战斗
  │     ├─ 胜利 → 奖励：随机稀有材料 + 道人指点（经验 +500）
  │     └─ 失败 → 无损失，道人留下「下次再来」后离开
  └─ 无视 → 道人摇头离去（无奖惩，下次遇见的概率降低）
```

## 八、奖励与声望

### 8.1 任务奖励公式

```
基础经验奖励 = 任务基础值 × 境界系数 × 连续日常倍率（仅日常）
灵石奖励     = 任务基础值 × 50 + 境界补正（每境 +20）
声望变化     = 固定值 ±5~±50（取决于选项）
装备奖励     = 任务固定 + 品质随机（良品~传说，取决于任务难度）
```

### 8.2 声望等级

| 等级 | 所需声望 | 解锁内容 |
|------|---------|---------|
| 冷淡 | <0 | 商店价格 +20%，部分 NPC 拒绝对话 |
| 中立 | 0~99 | 默认状态 |
| 友好 | 100~499 | 商店价格 -10%，解锁支线任务 |
| 尊敬 | 500~999 | 商店价格 -20%，解锁特殊商品 |
| 崇敬 | 1000~1999 | 商店价格 -30%，解锁专属功法/装备 |
| 崇拜 | ≥2000 | 商店价格 -50%，解锁隐藏剧情/结局线索 |
