# 界面设计原则

> 本章定义 UI 的整体设计风格与交互原则。

---

## 设计风格

| 维度 | 说明 |
|------|------|
| **风格** | 像素风 UI，融入古风/水墨元素 |
| **字体** | 像素风格中文字体（如 Fusion Pixel Font） |
| **字号** | 标题 16px，正文 12px，提示 8px |
| **主色调** | 米黄 `#f4ecd8`、墨色 `#2c2c2c`、朱红 `#c23b22` |
| **边框** | 仿古书卷轴边框，2px 内发光 |
| **图标** | 16×16 像素图标 |

---

## 交互原则

| 原则 | 说明 |
|------|------|
| **一致性** | 所有界面使用统一组件库 |
| **反馈即时** | 每次操作 → 0.1s 内视觉/音效反馈 |
| **层级清晰** | 最多 3 层界面堆叠（HUD → 面板 → 弹窗） |
| **键盘友好** | 所有界面支持键盘导航（Tab / 方向键 / 快捷键） |
| **可配置** | 支持 UI 缩放、HUD 自定义布局 |

---

## 界面层级

```
Layer 0 (最底层) — 游戏世界渲染
Layer 1 — HUD（血量/灵力/小地图/快捷键）
Layer 2 — 面板（背包/任务/地图/制作）
Layer 3 — 弹窗（确认/对话/商店）
Layer 4 (最顶层) — 暂停菜单 / 过场动画
```

---

## 色彩与无障碍

- 关键信息不使用**仅颜色**区分（同时使用图标/文字）
- 支持色盲模式（图标优先于颜色编码）
- 字体大小可在设置中调整
