## HUD 元素

### 顶部状态栏

| 元素 | 说明 |
|------|------|
| **HP 条** | 红色，当前生命/最大生命 |
| **MP 条** | 蓝色，当前灵力/最大灵力 |
| **SP 条** | 绿色，当前体力/最大体力 |
| **境界/等级** | 显示当前境界 + 等级 |
| **区域名** | 当前所在区域 |
| **灵石/灵玉** | 当前持有的货币数量 |

### 小地图

- 右上角缩略地图
- 显示：玩家位置（箭头）、已探索区域（亮色）、未探索（暗色）、任务目标（标记）
- 缩放级别 ×1 / ×2 / ×4

### 快捷键栏

- 底部 6 个快捷键槽
- 可绑定：符箓、古篆文、丹药、武器
- 显示快捷键位（Q/E/R/F/C/V）
- 冷却中显示遮罩 + 冷却数字
- 快捷键栏可展开为完整技能书

## HUD 特殊情况

| 情况 | HUD 变化 |
|------|----------|
| 战斗中 | HP/MP 条变亮，显示伤害数字 |
| 对话中 | HUD 半透明淡化 |
| 菜单打开 | HUD 隐藏 |
| 过场动画 | HUD 完全隐藏 |
| 低血量 | HP 条闪烁 + 屏幕边缘泛红 |
| 灵力充沛 | MP 条周围出现光晕 |
