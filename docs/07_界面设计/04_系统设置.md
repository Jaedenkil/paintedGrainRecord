# 系统设置

> 本章定义游戏的所有系统设置项、界面布局与交互逻辑，覆盖画面、音频、控制、游戏、无障碍五大模块。
>
> **参考游戏**：Minecraft（简洁布局）、Terraria（完整分类）、Hades/Celeste（无障碍标杆）、Stardew Valley（像素风UI缩放）、Godot 引擎（技术实现参考）

---

### 1.2 界面规范

| 元素 | 规范 | 说明 |
|:----:|------|------|
| **选项卡** | 5 个固定标签页，像素古风字体 14px | 参考 Minecraft 的简洁标签切换 |
| **分组标题** | 仿古卷轴分隔线，12px 加粗 | 每组设置之间用 1px 墨色分隔线 |
| **设置项** | 8px 行高，悬停时高亮 #c23b22 边框 | 参考 Terraria 的设置列表样式 |
| **滑块** | 像素风滑块，10px 宽，拖动时数值实时更新 | 音量/灵敏度等连续值 |
| **复选框** | □/☑ 像素风格，16×16 | 开关类设置 |
| **下拉菜单** | 仿古风卷轴下拉，最大显示 6 项 | 分辨率/语言等多项选择 |
| **按钮** | 像素风按钮 80×24，悬停放大效果 | 恢复默认/应用/取消 |

### 1.3 设置存储

| 存储类型 | 文件路径 | 生命周期 |
|---------|---------|---------|
| **本地配置** | `user://settings.cfg` | 跨存档共享 |
| **角色绑定** | 存档文件内 | 仅当前角色 |
| **会话临时** | 内存变量 | 当前游戏会话 |

> 所有设置修改后点击「应用」才写入文件，点击「取消」恢复修改前状态，点击「恢复默认」重置为初始值。

## 二、画面设置（Video/Graphics）

### 2.1 设置项总表

参考 **Minecraft** 的简洁画面设置 + **Stardew Valley** 的像素优化选项。

| 设置项 | 类型 | 默认值 | 选项范围 | 说明 |
|--------|:----:|:------:|---------|------|
| **显示模式** | 下拉 | 窗口 | 窗口 / 全屏 / 无边框窗口 | 切换显示模式（参考 Terraria 三种模式） |
| **分辨率** | 下拉 | 1920×1080 | 常见分辨率列表 + 当前桌面分辨率 | 仅在「窗口/无边框」模式下可选（参考 Minecraft） |
| **VSync** | 复选框 | ✔ 开 | 开/关 | 垂直同步，关闭可降低输入延迟但可能画面撕裂 |
| **FPS 上限** | 下拉 | 60 | 30 / 60 / 120 / 144 / 不限 | 帧率上限（参考 Godot 的 Engine.max_fps） |
| **UI 缩放** | 滑块 | 1.0x | 0.5x ~ 2.0x（步长 0.25） | 界面整体缩放（像素风保持整数缩放最佳） |
| **像素完美渲染** | 复选框 | ✔ 开 | 开/关 | 使用 `CanvasItem.texture_filter=NEAREST`，关闭则双线性过滤产生模糊（参考 Stardew Valley） |
| **屏幕震动强度** | 滑块 | 70% | 0% ~ 100%（步长 10%） | 受击/爆炸时的屏幕震动幅度（参考 Celeste 设置） |
| **粒子效果密度** | 下拉 | 高 | 关 / 低 / 中 / 高 | 法术特效/环境粒子的密度控制（性能关键） |
| **场景光照效果** | 复选框 | ✔ 开 | 开/关 | 关闭以提升性能（方块面法线光照等） |
| **后处理特效** | 复选框 | ✔ 开 | 开/关 | 泛光/色调映射等后期效果（像素风默认关闭） |
| **镜头平移平滑** | 复选框 | ✔ 开 | 开/关 | 镜头移动时的平滑插值，关闭则即时跳转 |
| **锯齿消除** | 下拉 | 2x | 关 / 2x / 4x | MSAA 抗锯齿（像素风建议关闭或 2x） |
| **渲染缩放** | 下拉 | 1.0x | 0.5x / 0.75x / 1.0x / 1.5x / 2.0x | 内部分辨率缩放（性能优化关键选项） |

## 三、音频设置（Audio）

### 3.1 设置项总表

参考 **Minecraft** 的多通道音量分离 + **Terraria** 的独立音效控制。

| **语音音量** | 滑块 | 100% | 0% ~ 100%（步长 5%） | NPC 对话/过场动画语音音量 |
| **环境音效** | 复选框 | ✔ 开 | 开/关 | 场景环境音（风声/水声/鸟鸣等） |
| **最小化时静音** | 复选框 | ☐ 关 | 开/关 | 窗口最小化时自动静音 |
| **音频输出设备** | 下拉 | 默认设备 | 系统可用音频设备列表 | 选择音频输出设备 |

### 4.3 键位冲突检测规则

| 场景 | 行为 |
|:----:|------|
| **新键位已被占用** | 弹出提示："按键 [X] 已被「技能1」占用，是否交换？"（参考 Godot InputMap 的冲突处理） |
| **新键位与系统键冲突** | 拒绝绑定，提示："按键 [F4] 是系统保留键" |
| **同一动作绑多键** | 支持：一个动作可绑定键盘+鼠标+手柄各一个 |
| **重置全部键位** | 确认弹窗后恢复所有键位为默认值 |

## 六、无障碍设置（Accessibility）

### 6.1 设计理念

参考 **Celeste** 的辅助模式（无障碍标杆） + **Hades** 的上帝模式 + **微软 Xbox 无障碍指南**。

> **目标**：在不改变核心游戏体验的前提下，让更多玩家能够享受《云汲仙田录》。

### 6.2 设置项总表

| 设置项 | 类型 | 默认值 | 选项范围 | 说明 |
|--------|:----:|:------:|---------|------|
| **色盲模式** | 下拉 | 关闭 | 关闭 / 红色盲(Protanopia) / 绿色盲(Deuteranopia) / 蓝色盲(Tritanopia) | 调整游戏内颜色编码为色盲友好方案（参考 Hades 色盲模式） |
| **高对比度模式** | 复选框 | ☐ 关 | 开/关 | 增强 UI 元素与背景的对比度（参考 Celeste） |
| **大字模式** | 下拉 | 关闭 | 关闭 / 仅UI / 全部（含对话） | 所有文本字体放大 1.5x（参考 Stardew Valley UI缩放） |
| **减少闪烁** | 复选框 | ☐ 关 | 开/关 | 禁用频闪/闪烁特效（癫痫安全——参考 Celeste） |
| **慢速模式** | 滑块 | 1.0x | 0.5x ~ 1.0x（步长 0.1） | 降低游戏速度（参考 Celeste 辅助模式） |
| **自动瞄准强度** | 滑块 | 0% | 0% ~ 100%（步长 10%） | 攻击时自动对准最近敌人的强度（参考 Hades 自动瞄准） |
| **简化操作模式** | 复选框 | ☐ 关 | 开/关 | 一键连招/自动连击/简化符箓绘制（参考 Celeste 辅助模式） |
| **无敌模式** | 复选框 | ☐ 关 | 开/关 | 角色不会死亡（探索/剧情向玩家，参考 Hades 上帝模式） |
| **无限灵力** | 复选框 | ☐ 关 | 开/关 | 灵力不会消耗（参考 Celeste 无限冲刺） |
| **字幕** | 复选框 | ✔ 开 | 开/关 | NPC 对话/过场动画字幕 |
| **音频提示** | 复选框 | ✔ 开 | 开/关 | 关键游戏事件（受击/拾取/升级）的额外音频提示 |
| **按键防抖时间** | 滑块 | 100ms | 0ms ~ 500ms（步长 50ms） | 防止按键连发的时间窗口（参考 Windows 粘滞键） |
| **界面朗读** | 复选框 | ☐ 关 | 开/关 | UI 元素的 TTS 语音朗读（视障玩家，需系统 TTS 支持） |

### 7.2 Godot 实现要点

```gdscript
# SettingsManager.gd — 设置管理（Autoload）
extends Node

# 设置变更信号
signal setting_changed(category: String, key: String, value: Variant)
signal settings_applied()
signal accessibility_changed()  # 无障碍设置变更时发射

const SETTINGS_PATH: String = "user://settings.cfg"

var config: ConfigFile = ConfigFile.new()

func _ready() -> void:
    load_settings()

# 加载设置
func load_settings() -> void:
    if FileAccess.file_exists(SETTINGS_PATH):
        config.load(SETTINGS_PATH)
    else:
        set_defaults()
    apply_all()

# 应用所有设置
func apply_all() -> void:
    apply_video_settings()
    apply_audio_settings()
    apply_control_settings()
    apply_gameplay_settings()
    apply_accessibility_settings()
    settings_applied.emit()

# 获取设置值（带默认值回退）
func get_setting(category: String, key: String, default: Variant) -> Variant:
    if config.has_section_key(category, key):
        return config.get_value(category, key, default)
    return default

# 设置值
func set_setting(category: String, key: String, value: Variant) -> void:
    config.set_value(category, key, value)
    setting_changed.emit(category, key, value)

# 保存到文件
func save_settings() -> void:
    config.save(SETTINGS_PATH)

# 恢复默认
func reset_to_defaults() -> void:
    var backup_path: String = SETTINGS_PATH + ".bak"
    if FileAccess.file_exists(SETTINGS_PATH):
        DirAccess.rename_absolute(SETTINGS_PATH, backup_path)
    set_defaults()
    apply_all()

# 应用画面设置
func apply_video_settings() -> void:
    var mode: String = get_setting("graphics", "display_mode", "windowed")
    match mode:
        "windowed":
            DisplayServer.window_set_mode(DisplayServer.WINDOW_MODE_WINDOWED)
        "fullscreen":
            DisplayServer.window_set_mode(DisplayServer.WINDOW_MODE_FULLSCREEN)
        "borderless":
            DisplayServer.window_set_mode(DisplayServer.WINDOW_MODE_WINDOWED)
            DisplayServer.window_set_flag(DisplayServer.WINDOW_FLAG_BORDERLESS, true)
    
    # FPS 上限
    Engine.max_fps = get_setting("graphics", "fps_cap", 60)
    
    # VSync
    DisplayServer.window_set_vsync_mode(
        DisplayServer.VSYNC_ENABLED if get_setting("graphics", "vsync", true) \
        else DisplayServer.VSYNC_DISABLED
    )
    
    # 像素完美渲染
    if get_setting("graphics", "pixel_perfect", true):
        get_tree().root.texture_filter = CanvasItem.TEXTURE_FILTER_NEAREST
    else:
        get_tree().root.texture_filter = CanvasItem.TEXTURE_FILTER_LINEAR

# 应用无障碍设置
func apply_accessibility_settings() -> void:
    var colorblind: String = get_setting("accessibility", "colorblind_mode", "off")
    # 色盲模式通过 Shader 实现颜色变换
    match colorblind:
        "protanopia":
            # 应用红色盲滤镜 Shader
            pass
        "deuteranopia":
            # 应用绿色盲滤镜 Shader
            pass
        "tritanopia":
            # 应用蓝色盲滤镜 Shader
            pass
        _:
            # 移除滤镜
            pass
    
    # 大字模式
    var large_font: String = get_setting("accessibility", "large_font", "off")
    # 通过 Theme 的字体大小动态调整
    match large_font:
        "ui_only":
            # 仅放大 UI 字体
            pass
        "all":
            # 放大所有文本
            pass
    
    accessibility_changed.emit()
```
