# 背包与物品栏

> 本章定义背包界面与物品管理交互。

---

## 背包界面

```
┌──────────────────────────────────────┐
│  [背包]  [装备]  [符箓]  [古篆文]    │ ← 选项卡
├──────────────────────────────────────┤
│                                      │
│  ┌────┐ ┌────┐ ┌────┐ ┌────┐ ┌────┐│
│  │灵药│ │材料│ │丹方│ │灵种│ │ 空 ││
│  ├────┤ ├────┤ ├────┤ ├────┤ ├────┤│
│  │x12 │ │x5  │ │x1  │ │x3  │ │    ││
│  └────┘ └────┘ └────┘ └────┘ └────┘│
│  ┌────┐ ┌────┐ ┌────┐ ┌────┐ ┌────┐│
│  │    │ │    │ │    │ │    │ │    ││
│  │    │ │    │ │    │ │    │ │    ││
│  └────┘ └────┘ └────┘ └────┘ └────┘│
│                                      │
│  灵石: 1240    负重: 12/40           │
├──────────────────────────────────────┤
│  [选中物品信息面板]                    │
│  火灵花 x3                           │
│  用途: 炼丹材料 / 制符材料            │
│  说明: 蕴含火属性灵气的奇特花朵        │
│                                      │
│  [使用] [丢弃] [拆分]                 │
└──────────────────────────────────────┘
```

---

## 物品分类

| 分类 | 说明 | 堆叠上限 |
|------|------|----------|
| 灵药 | 炼丹产物，可消耗 | 20 |
| 材料 | 灵材、矿石、兽皮 | 99 |
| 丹药 | 成品丹药 | 10 |
| 符箓 | 成品符箓（可装备到快捷键） | 20 |
| 古篆文 | 古篆文字符碎片/完整 | 1（唯一） |
| 装备 | 武器、衣袍、饰品、法宝 | 1（唯一） |
| 灵种 | 灵田种植用种子 | 99 |
| 任务 | 任务关键物品 | 1（唯一） |
| 杂项 | 灵石、钥匙、卷轴 | 99 |
