# 交互反馈

> 本章定义游戏中的提示、对话框与通知系统。

---

## 通知系统

### 消息类型

| 类型 | 颜色 | 显示位置 | 持续 | 示例 |
|------|------|----------|------|------|
| 信息 | 白色 | 左上角 | 3s | "获得 火灵花 ×3" |
| 成功 | 绿色 | 左上角 | 3s | "炼丹成功！获得 培元丹 ×2" |
| 警告 | 黄色 | 中央偏下 | 4s | "灵力不足，无法释放！" |
| 错误 | 红色 | 中央偏下 | 4s | "炼丹失败……" |
| 任务 | 金色 | 左上角特殊 | 5s | "主线任务更新：前往青竹谷" |
| 成就 | 紫色 | 中央 | 6s | "成就解锁：初次突破！" |

### 消息队列
- 同时最多显示 3 条消息
- 新消息从上方滑入，旧消息向下滚动
- 点击可关闭单条消息

---

## 交互提示

| 类型 | 外观 | 触发条件 |
|------|------|----------|
| NPC 头顶提示 | "..." / "?" / "!" / "✦" | 可交互 NPC |
| 可交互物品 | 物品高亮 + 闪烁 | 玩家靠近 |
| 可采集资源 | 采集标记 | 玩家靠近资源点 |
| 任务标记 | 三角箭头 + 方向 | 指向任务目标 |
| 危险标记 | 红色骷髅 | 高威胁敌人 |
| 传送点 | 蓝色漩涡标记 | 已激活传送点 |

## 对话框

### 确认弹窗

```
┌──────────────────────────┐
│      保存游戏             │
│                          │
│  是否保存当前进度？        │
│                          │
│  [确定]      [取消]      │
└──────────────────────────┘
```

### 选择弹窗（随机事件）

```
┌──────────────────────────┐
│      ⚠ 遭遇妖兽群        │
│                          │
│  前方发现大量妖兽足迹！   │
│                          │
│  [战斗] 正面迎战          │
│  [绕行] 消耗更多时间但安全 │
│  [撤退] 返回上个区域      │
└──────────────────────────┘
```

## 加载/保存提示

| 界面 | 表现 |
|------|------|
| 场景加载 | 像素风 Loading 动画 + 随机小贴士 |
| 自动存档 | 右上角小图标闪烁 1s（"📀"） |
| 手动保存 | 保存弹窗显示 2s 后自动关闭 |
| 读取存档 | 进度条 + "正在唤醒灵田……" |
