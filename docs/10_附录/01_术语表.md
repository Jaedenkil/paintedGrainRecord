# 术语表

> 本章列出游戏设计文档中使用的专业术语与修仙术语解释。

---

## 游戏术语

| 术语 | 说明 |
|------|------|
| **GDD** | Game Design Document，游戏设计文档 |
| **HUD** | Heads-Up Display，平视显示器 |
| **BGM** | Background Music，背景音乐 |
| **SFX** | Sound Effects，音效 |
| **NPC** | Non-Player Character，非玩家角色 |
| **Boss** | 关卡/区域的最终强敌 |
| **Hitbox** | 攻击判定框 |
| **Hurtbox** | 受击判定框 |
| **HitStop** | 命中停帧，提升打击感的技术 |
| **TileMap** | 瓦片地图 |
| **TileSet** | 瓦片集 |
| **Atlas** | 图集（多个小图拼成的大图） |
| **SpriteSheet** | 精灵表（多个动画帧拼成的图片） |
| **Autoload** | Godot 中的全局单例 |
| **SignalBus** | 信号总线，模块间通信的中间件 |
| **InputMap** | Godot 的输入映射系统 |
| **Roguelite** | 轻度 Roguelike，带有永久成长的随机探索游戏 |

## 属性体系术语（天道九层框架）

| 术语 | 说明 |
|------|------|
| **力量 (STR)** | 二气·阴阳层的基础战斗属性，决定物理攻击与近战伤害加成 |
| **耐力 (CON)** | 二气·阴阳层的基础防御属性，决定生命值与物理防御加成 |
| **神识 (INT)** | 三才·天位属性，决定法术攻击与灵力值上限，法系流派核心 |
| **体魄 (END)** | 三才·地位属性，决定生命值上限与全伤害减免，坦克流派核心 |
| **身法 (DEX)** | 三才·人位属性，决定暴击率/闪避/攻速/移速，敏捷流派核心 |
| **猛攻 (ATK)** | 四象·太阳位战斗属性，最终物理攻击力，受 STR × 物种系数影响 |
| **铁壁 (DEF)** | 四象·太阴位战斗属性，最终物理防御力，受 CON × 物种系数影响 |
| **治愈 (HEAL)** | 四象·少阳位战斗属性，治疗效果加成，受 INT × 物种系数影响 |
| **急速 (SPD)** | 四象·少阴位战斗属性，攻击速度与移动速度，受 DEX × 物种系数影响 |
| **福缘** | 九宫层命理属性，影响稀有物品掉落率和随机事件正面结果概率 |
| **天时** | 七星层周期属性，反映当前时辰/七曜日对行为和战斗的加成方向 |
| **空间掌控** | 六合层空间属性，决定技能范围、传送效率和位移能力 |
| **无视防御** | 六合层攻击属性，攻击时直接穿透目标防御的比例，最高 80% |
| **八卦特效强度** | 八卦层法则属性，决定八卦特效（击退/眩晕/灼烧/冰冻等）的触发几率和效果 |
| **八卦特效抗性** | 八卦层法则属性，抵抗敌方八卦特效的概率和效果减免 |
| **九宫精通** | 九宫层命理属性，影响炼丹成功率、布阵效果和物品鉴定品质 |
| **混元** | 一元层本体属性，0.0~1.0 混沌度，影响所有属性的随机波动幅度 |

## 战斗属性术语

| 术语 | 说明 |
|------|------|
| **二级属性** | 由基础属性推导的衍生战斗属性（暴击率、穿透、闪避、格挡等） |
| **暴击率** | 攻击造成暴击的概率，基础 3%，受灵敏和装备影响 |
| **暴击伤害** | 暴击时伤害倍率，基础 150%，上限由装备/经脉提供 |
| **命中率** | 攻击命中目标的概率，基础 95%，受闪避率影响 |
| **穿透** | 无视目标防御的比例，最高 80%，由装备和技能提供 |
| **破甲** | 命中后降低目标防御值的比例 |
| **格挡率** | 减少受到伤害的概率（伤害减半），由装备和功法提供 |
| **韧性** | 减少被暴击的概率和暴击伤害，与暴击率互相抵消 |
| **冷却缩减** | 技能冷却时间减少比例，上限 40% |
| **五行亲和** | 对应五行属性的伤害加成，每点 +2%，50 点后衰减 |
| **五行抗性** | 对应五行属性的伤害减免，每点 -1.5%，上限 75 点 |
| **五行掌握** | 对应五行特殊效果触发率，每点 +1%，上限 60% |
| **五行相克** | 金→木→土→水→火→金的克制循环，克制时 ×1.5，被克时 ×0.75 |
| **五行相生** | 金→水→木→火→土→金的增益循环，命中后给下一系技能加伤 |
| **境界系数** | 基于境界差的伤害修正，每高一大境界 +0.1，每低一大境界 -0.1 |
| **属性收益衰减** | 属性堆叠超过阈值后效果减半/四分之一的分段衰减机制 |
| **法术熟练度** | 法术使用次数带来的等级提升，Lv.1~5 逐级减耗增伤减冷却 |
| **法术强化分支** | 法术宗师级（Lv.5）时选择的二选一永久强化方向 |
| **基础剑式** | 剑诀的四种基本攻击动作（刺/劈/扫/挑），无冷却消耗体力 |
| **连段剑诀** | 按特定顺序输入基础剑式触发的组合技，有冷却时间 |
| **蓄力爆发技** | 长按攻击键蓄力后释放的强力剑技，蓄力时间越长伤害越高 |
| **组合符箓** | 在 0.5s 内连续绘制两个基础符箓触发的五行连携法术 |
