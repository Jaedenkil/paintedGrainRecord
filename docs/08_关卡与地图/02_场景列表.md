# 场景列表

> 本章列出所有场景的详细设计，基于 **2.5D 方块系统** 的网格坐标与方块配置。

---

## 场景结构

每个场景包含以下要素：

| 元素 | 说明 |
|------|------|
| 场景 ID | 唯一标识，如 `scene_valley_01` |
| 名称 | 中文名 |
| 类型 | 野外 / 副本 / 城镇 / 室内 |
| **方块网格尺寸** | `gx × gy` 网格数（替换旧 Tile 尺寸） |
| **最大高度** | `max_gz` 场景最高允许层数 |
| 主方块类型 | 场景使用的主要方块种类 |
| 敌人 | 出现的敌人类型与密度 |
| 资源 | 可采集的资源 |
| 连接 | 相邻场景 |
| 光照 | 场景光源参数 |
| 特殊 | 该场景的特殊机制/事件 |

## 场景光照参数一览

| 场景 | 主光源类型 | 光源颜色 | 强度 | 夜晚循环 | 局部光源 |
|------|-----------|---------|------|---------|---------|
| 灵田 | DirectionalLight2D | `#8fc0a9` | 1.0 | ❌ | 聚灵阵发光(闪烁) |
| 青竹谷 | DirectionalLight2D | `#8fc0a9` | 0.9~1.0 | ✅ 昼/夜 | 瀑布水光 |
| 落霞山脉 | DirectionalLight2D | `#e76f51` | 0.8 | ❌ 恒定黄昏 | 矿洞入口发光 |
| 幽暗密林 | DirectionalLight2D | `#264653` | 0.4 | ✅ 夜晚更暗 | 萤火虫(6~10个PointLight2D) |
| 碧波湖泽 | DirectionalLight2D | `#4a8fc0` | 0.8 | ✅ | 水面反射光 |
| 炎火深渊 | 无全局光 | — | — | ❌ | 岩浆自发光(15+ PointLight2D) |
| 万雷天谷 | DirectionalLight2D | `#6b4c85` | 0.6 | ❌ 恒定 | 闪电脉冲光 |

> **光照对方块的影响**：所有光源通过 Godot `Light2D` 节点作用于方块精灵。三个面对光照的响应不同（上平面最敏感、垂直面较暗），通过方块面法线方向模拟 3D 光照效果。
