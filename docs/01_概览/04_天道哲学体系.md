> **一元混元 → 二气阴阳 → 三才天地人 → 四象太阳太阴少阳少阴 → 五行金木水火土 → 六合天地四方 → 七星日月五曜 → 八卦乾坤震巽坎离艮兑 → 九宫定数**

## 一、一元·混元 — 万物之始

### 1.1 哲学本义

> **「道生一，一生二，二生三，三生万物。」** ——《道德经》

**混元** 又称 **混沌**、**太极（未分状态）**，是宇宙诞生之前的原始状态。它无形无象、无阴无阳、无善无恶，却蕴含一切可能的种子。

在《云汲仙田录》的世界中，混元并非"不存在"，而是 **「万有之潜能」**——一切法则、一切力量、一切生命的根源。

### 1.2 修仙诠释

| 层面 | 混元体现 | 说明 |
|------|----------|------|
| **宇宙本源** | 混沌元气 | 开天辟地之前的原始能量，至今仍存在于混沌秘境中 |
| **修士根基** | 先天一炁 | 修士出生时带有的那一缕混元之气，是修行的根本 |
| **最高境界** | 混元道果 | 登仙之后的终极境界，与道合真，返归混元 |
| **古篆文** | 一阶·混沌 9 篆 | 108 篆之首，代表天地最本源的九种法则力量 |
| **功法** | 混元功 | 传说级功法，直指混元大道 |

### 1.3 游戏映射

| 系统 | 混元应用 |
|------|----------|
| **修炼** | 「混元道果」为登仙后的隐藏内容，九枚混沌篆全收集解锁「混沌归一」终极被动 |
| **战斗** | 混沌属性伤害 = 真实伤害（无视五行抗性和防御） |
| **炼丹** | 「混沌丹」— 无属性，可转化为任何五行属性 |
| **炼器** | 「混沌石」— 传说级材料，可嵌入任何装备提升全属性 |
| **地图** | 「混沌秘境」— 游戏后期隐藏区域，法则混乱，五行不定 |
| **剧情** | 最终 BOSS 可能掌握混元之力，是天道失衡的产物 |

### 1.4 混元在九层中的位置

混元是九层之 **体**，其余八层是混元之 **用**。在游戏数值体系中，混元对应 **第七阶**（超越六阶的存在），只有登仙后的内容触及。

## 二、二气·阴阳 — 两仪初分

### 2.1 哲学本义

> **「太极生两仪，两仪生四象。」** ——《周易·系辞》

混元一动，清轻者上升为 **阳**，重浊者下沉为 **阴**。阴阳是宇宙最基本的两种对立统一力量：

| 阴阳属性对比 | 阳 | 阴 |
|-------------|-----|------|
| **性质** | 刚健、主动、向外、上升 | 柔顺、被动、向内、下沉 |
| **天象** | 日、昼、晴、夏 | 月、夜、雨、冬 |
| **时间** | 子时→午时（阳气渐长） | 午时→子时（阴气渐长） |
| **人体** | 腑（六腑）、表、背 | 脏（五脏）、里、腹 |
| **修仙** | 金丹、阳神、火候 | 元婴、阴神、温养 |
| **数字** | 奇数（1,3,5,7,9） | 偶数（2,4,6,8,10） |

### 2.2 阴阳互根与转化

阴阳不是绝对的二元对立，而是 **相互依存、相互转化**：

```
阴中有阳，阳中有阴
阴极生阳，阳极生阴
孤阴不生，独阳不长
```

这体现在游戏中：
- **昼夜循环**：白天阳气盛（火/金属性增强），夜晚阴气盛（水/木属性增强），子午交替时阴阳平衡
- **功法阴阳**：阳属性功法在白昼威力 +15%，阴属性功法在夜间威力 +15%
- **丹药阴阳**：阳丹提升攻击/暴击，阴丹提升防御/恢复
- **渡劫阴阳**：雷劫为阳，心魔劫为阴

### 2.3 游戏映射

| 系统 | 阴阳应用 |
|------|----------|
| **时间系统** | 12 时辰循环，每个时辰阴阳消长不同，影响修炼效率 |
| **经脉** | 十二正经中六条阳经（手三阳+足三阳）、六条阴经（手三阴+足三阴） |
| **符箓** | 阳符（攻击型）与阴符（辅助/控制型）之分 |
| **功法** | 阳属性功法（爆发输出）vs 阴属性功法（持续续航） |
| **丹药** | 阳丹（红色）增益攻击/暴击；阴丹（蓝色）增益防御/恢复 |
| **阵法** | 阳阵主杀伐，阴阵主困守，阴阳阵兼备 |
| **修炼** | 子时（阴之极）打坐修阴神，午时（阳之极）打坐修阳神 |
| **风水** | 阳宅（洞府）选址需背阴向阳，阴穴（秘境入口）多在背阳处 |

### 2.4 阴阳在九层中的位置

阴阳是九层之 **用**，是混元动起来之后的第一重分化。九层体系从阴阳开始有了"对偶"结构——之后的四象、八卦皆为阴阳的进一步分化。

## 三、三才·天地人 — 三极之道

### 3.1 哲学本义

> **「立天之道曰阴与阳，立地之道曰柔与刚，立人之道曰仁与义。」** ——《周易·说卦》

阴阳交感之后，清气上升为 **天**，浊气下沉为 **地**，天地交合之气化生为 **人**（万物之灵）。三才并立，构成世界的基本框架。

| 三才 | 象征 | 法则 | 属性 |
|------|------|------|------|
| **天** | 乾、阳、刚健 | 天道运行，四时更替 | 时间法则 |
| **地** | 坤、阴、柔顺 | 地道承载，万物生长 | 空间法则 |
| **人** | 中、和、灵明 | 人道修行，参赞化育 | 意识法则 |

### 3.2 三才之道

**三才的核心在于「人」的特殊地位**——人处于天地之间，既能上察天时，又能下接地利，通过修行可以 **「参赞天地之化育」**，即参与并改造天地的运行。

这在游戏中的体现：
- **天时系统**：时辰、天气、季节影响修炼和种植
- **地利系统**：地形、灵气浓度、风水影响洞府和阵法
- **人和系统**：技能选择、经脉加点、流派搭配决定战斗风格
- **三才合一**：天时+地利+人和三者配合时，效率最大化

### 3.3 游戏映射

| 系统 | 三才应用 |
|------|----------|
| **修炼** | 天（境界突破时机）+ 地（洞府灵气浓度）+ 人（自身根基）三者决定突破成功率 |
| **灵田** | 天时（季节/天气）+ 地利（土壤/灵脉）+ 人和（耕作技能）决定作物品质 |
| **阵法** | 三才阵：阵眼=天，阵基=地，阵脚=人，三者联动时效果翻倍 |
| **战斗** | 天（五行生克）+ 地（地形因素）+ 人（操作/技能搭配）决定战局 |
| **炼丹** | 天（七曜时辰）+ 地（丹炉/地火）+ 人（控火/配方）决定丹药品质 |
| **古篆文** | 天篆（自然法则类）+ 地篆（生物灵兽类）+ 人篆（抽象概念类）三分类 |

### 3.4 三才在九层中的位置

三才是九层之 **结构**。从三才开始，天道有了"三层框架"，之后的四象、五行、六合等都在这个框架内填充。

## 四、四象·太阳太阴少阳少阴 — 四维立象

### 4.1 哲学本义

> **「两仪生四象。」** ——《周易·系辞》

阴阳再分，各生两面：

```
         ┌─── 太阳（阳中之阳）── 夏 ── 南 ── 火
         │
   阳 ── ┤─── 少阳（阴中之阳）── 春 ── 东 ── 木
         │
   阴 ── ┤─── 少阴（阳中之阴）── 秋 ── 西 ── 金
         │
         └─── 太阴（阴中之阴）── 冬 ── 北 ── 水
```

### 4.2 四象对应表

| 四象 | 含义 | 季节 | 方位 | 五行 | 四灵 | 属性 | 颜色 |
|------|------|------|------|------|------|------|------|
| **太阳** | 阳之极致 | 夏 | 南 | 火 | 朱雀 | 攻击 | 赤 |
| **少阳** | 阴中初阳 | 春 | 东 | 木 | 青龙 | 生命/成长 | 青 |
| **少阴** | 阳中初阴 | 秋 | 西 | 金 | 白虎 | 暴击/肃杀 | 白 |
| **太阴** | 阴之极致 | 冬 | 北 | 水 | 玄武 | 防御/守护 | 黑 |

额外：**中土**（季夏/中央/土/黄麟）为四象之枢纽，但不属于四象本身。

### 4.3 游戏映射

| 系统 | 四象应用 |
|------|----------|
| **战斗属性** | 太阳→攻击、太阴→防御、少阳→生命/恢复、少阴→暴击/速度 |
| **四灵套装** | 青龙套装（木系持续）、白虎套装（金系暴击）、朱雀套装（火系爆发）、玄武套装（水系防御） |
| **四象阵法** | 青龙阵（生命恢复）、白虎阵（攻击加成）、朱雀阵（灵力爆发）、玄武阵（护盾减免） |
| **四季系统** | 春（木旺、种植最佳）、夏（火旺、炼丹最佳）、秋（金旺、炼器最佳）、冬（水旺、修炼最佳） |
| **四象秘境** | 青龙秘境（木系素材）、白虎秘境（金系装备）、朱雀秘境（火系丹药）、玄武秘境（水系功法） |
| **四灵守护** | 集齐四灵古篆文（龙+凤+虎+龟）→ 四灵护体：全抗性 +20% |
| **四象招式** | 剑诀的四种基础剑式对应四象：刺（太阳/快）、扫（少阳/广）、劈（少阴/重）、挑（太阴/巧） |

### 4.4 四象在九层中的位置

四象是九层之 **框架**。它为世界搭建了四个基本维度，之后的五行、六合、八卦都在四象的基础上进一步细化。

五行是构成万物的五种基本元素（材质），也是五种相互作用的力量。五行体系已在现有游戏设计中广泛应用，此处从哲学高度做系统性梳理。

## 六、六合·天地四方 — 空间整体

### 6.1 哲学本义

> **「六合之外，圣人存而不论；六合之内，圣人论而不议。」** ——《庄子·齐物论》

**六合** 指 **上、下、东、西、南、北** 六个方向，即 **三维空间的全部六个维度**。在修仙世界观中，六合代表 **"天地四方之内"**——即整个已知宇宙的范围。

| 六合 | 方位 | 对应四象 | 空间意义 |
|------|------|----------|----------|
| **上（天）** | 天穹 | 太阳 | 上升、飞跃、超越 |
| **下（地）** | 地底 | 太阴 | 下沉、埋藏、根基 |
| **东** | 日出之方 | 少阳 | 新生、开始、春 |
| **西** | 日落之方 | 少阴 | 终结、收敛、秋 |
| **南** | 阳面 | 太阳 | 炽盛、光明 |
| **北** | 阴面 | 太阴 | 幽深、暗藏 |

### 6.2 六合在修仙中的意义

六合不仅是空间概念，更是 **阵法、封印、遁术** 的基础：

- **六合封印**：封锁上·下·东·西·南·北六个维度，使封印对象无法从任何方向逃脱
- **六合步法**：步踏六合方位，身法飘忽不定，是顶级身法的理论基础
- **六合空间**：储物法宝的内部空间=六合之内自成天地

### 6.3 游戏映射

| 系统 | 六合应用 |
|------|----------|
| **传送阵** | 利用六合空间法则，在六合内的两个坐标点之间建立空间通道 |
| **储物空间** | 高阶储物装备开辟"六合空间"，容量远超普通储物袋 |
| **六合封印术** | 封印类法术的基础理论，六方向同时封锁，破解需同时击破六处封印点 |
| **身法** | 六合步：向六个方向任意瞬移（冷却独立计算） |
| **阵法** | 六合阵：覆盖范围=以阵眼为中心的六面体区域，比平面阵法更立体 |
| **洞府风水** | 洞府选址讲究"六合藏风"——上承天光、下接地脉、四面环顾、气不外泄 |
| **战斗** | 六合范围攻击：同时攻击角色周围上·下·左·右·前·后六个方向 |

### 6.4 六合在九层中的位置

六合是九层之 **空间**。六合的出现标志着天道从"元素构成"（五行）进入了"空间维度"。之后的七星则引入了时间维度，从空间→时空。

## 七、七星·日月与五曜 — 天象节律

### 7.1 哲学本义

> **「天有七曜，地有五行。」** ——《尚书考灵曜》

**七星** 又称 **七政**、**七曜**，指 **日、月 与 金、木、水、火、土 五颗行星**。七曜的运行规律构成了天上世界的基本节律。

| 七曜 | 别称 | 星期对应 | 五行属性 | 象征 |
|------|------|----------|----------|------|
| **日（太阳）** | 太阳星 | 日曜日（Sunday） | 火之极致 | 阳刚、生机、光明 |

### 9.5 九宫炼丹详解（示例）

九宫炼丹是九宫体系在游戏中最深度的应用之一：

```
        ┌─────┬─────┬─────┐
        │  4  │  9  │  2  │
        │ 辅药 │ 佐药 │ 辅药 │  ← 上方三格对应头部（上焦）
        ├─────┼─────┼─────┤
        │  3  │  5  │  7  │
        │ 辅药 │ 主药 │ 辅药 │  ← 中间三格对应胸部（中焦）
        ├─────┼─────┼─────┤
        │  8  │  1  │  6  │
        │ 引药 │ 药引 │ 引药 │  ← 下方三格对应腹部（下焦）
        └─────┴─────┴─────┘
```

**放置规则**：
1. **中宫（5）**：必放主药，决定丹药基本类型
2. **坎宫（1）**：放药引，影响丹药品质下限
3. **离宫（9）**：放佐药，影响丹药特殊效果
4. **其他六宫**：放辅药，影响丹药数量/品质/特效
5. **五行相生放置**：相邻宫格若按五行相生顺序放置 → 药效 +20%
6. **九宫合丹**：九格放满且五行关系正确 → 必定传说品质 + 额外特效

| **九宫大阵** | 九宫 | 终极法术，需九宫布阵 |
| **混沌法术** | 一元 | 无视五行/防御的终极法则伤害 |

## 十一、九层体系在游戏中的实际应用原则

### 11.1 层层衍生，不越级

九层体系的设计遵循 **层层衍生** 原则：
- 修士从五行（第五层）开始感悟天道，逐步向上一元追溯、向下一九宫深入
- 凡人期感应一元（先天潜能）、炼气期接触阴阳、筑基期体悟三才、金丹期凝练五行、元婴期运用四象、化神期沟通八卦、悟道期参透九宫、羽化期掌握六合、登仙期回归混元——每层对应一个修炼阶段
- 不能跳级：未悟五行则无法理解八卦，未通八卦则无法参透九宫

### 11.2 九层不是九套独立系统

九层是一体九面，而非九个独立系统：
- **五行**是基础（已有），**八卦**是五行的八种组合方式
- **四象**是阴阳的进一步分化，**六合**是四象的空间化
- **七星**是五行+日月的时间化，**九宫**是八卦+中宫的数理化
- **混元**是一切的本体

### 11.3 渐进解锁

九层体系对玩家是 **渐进解锁** 的：
- 凡人期：感应混元（先天一炁，修行根基）
- 炼气期：感知阴阳（昼夜系统）
- 筑基期：体悟三才（天地人影响修炼）
- 金丹期：掌握五行（五行深度战斗）
- 元婴期：运用四象（四灵/四季系统）
- 化神期：沟通八卦（八卦阵/符）
- 悟道期：参透九宫（九宫炼丹/阵法）
- 羽化期：掌握六合（空间传送/跨越）
- 登仙后：回归混元（隐藏内容）
