# 游戏特色

> 本章列出《云汲仙田录》的核心卖点与差异化亮点。

---

## 核心卖点

### ✦ 符箓战斗系统
- 数十种符箓，分为 **金木水火土** 五行属性
- 符箓可组合释放，产生连携效果（如「水符 + 雷符 → 雷瀑术」）
- 战斗中实时绘制符箓，考验操作与策略

### ✦ 古篆字符文系统
- 借鉴甲骨文/金文的古篆字，每个字代表一种基础法则（如「火」「山」「风」「雷」）
- 古篆文可镶嵌到装备中，激活被动/主动效果
- 收集古篆文本身就是一条核心成长线

### ✦ 灵田经营
- 真实时间 / 游戏内时间驱动的种植系统
- 灵田可升级：扩大面积、改善灵脉、布置聚灵阵
- 种植的灵药是炼丹、炼器的核心原料

### ✦ 完整的修仙境界体系
- 凡人 → 炼气 → 筑基 → 金丹 → 元婴 → 化神 → 悟道 → 羽化 → 登仙
- 每个境界解锁新能力、新区域、新配方

---

## 差异化亮点

| 对比传统修仙游戏 | 《云汲仙田录》 |
|-----------------|----------------|
| 文字挂机式修炼 | 动作战斗 + 实时操作 |
| 简单合成 | 符箓 + 古篆文深度组合系统 |
| 静态场景 | 像素风格动态世界 + 天气系统 |
| 单一战斗方式 | 符箓远程 + 古篆文法阵 + 近战剑诀三修 |
