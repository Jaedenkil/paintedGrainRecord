# 核心概念

> **本章定位：** 用一页纸说清楚《云汲仙田录》是什么。

---

## 一句话描述

**《云汲仙田录》** 是一款以 **仙侠世界** 为背景的 **2D 像素风动作角色扮演游戏**。玩家扮演一名初入修仙之路的散修，在灵田中耕作、炼丹炼器增强实力，同时以符箓与古篆文为武器，探索秘境、斩妖除魔，最终突破境界、登仙成道。

---

## 核心玩法循环

```
探索副本/秘境
     ↓
    战斗（符箓 + 古篆文技能）
     ↓
获取资源（灵材、灵石、功法残卷）
     ↓
    经营灵田（种植灵药）
     ↓
    炼丹 / 炼器 / 绘制符箓
     ↓
提升修为 → 突破境界 → 解锁更强能力
     ↓
    挑战更高难度区域
     ↑_________________|
```

---

## 目标受众

| 维度 | 描述 |
|------|------|
| **年龄** | 16–35 岁 |
| **偏好** | 像素艺术、仙侠/修仙题材、动作角色扮演 |
| **游戏经验** | 熟悉 2D 动作游戏（如《死亡细胞》、《Hades》等） |
| **平台** | PC（Steam / itch.io） |

---

## 核心体验目标

1. **爽快的战斗** — 符箓与古篆文组合出丰富连招，打击感强
2. **真实的修仙成长** — 从凡人到登仙的完整境界体系
3. **经营的成就感** — 从一块荒地到灵田满园的养成乐趣
4. **深度的自定义** — 自由组合技能、装备、丹药打造独特流派
