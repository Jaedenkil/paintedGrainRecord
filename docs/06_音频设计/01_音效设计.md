# 音效设计

> 本章定义游戏中所有音效的类型与设计方向。

---

## 音效分类

### 环境音效

| 场景 | 音效 | 循环 |
|------|------|------|
| 青竹谷 | 竹林风声、鸟鸣、溪流 | ✅ |
| 灵田 | 虫鸣、微风、作物生长音 | ✅ |
| 落霞山脉 | 山风、矿石敲击声 | ✅ |
| 幽暗密林 | 猫头鹰叫声、树叶沙沙声 | ✅ |
| 碧波湖泽 | 水波、雨声、蛙鸣 | ✅ |
| 炎火深渊 | 岩浆翻滚、火焰燃烧 | ✅ |
| 万雷天谷 | 雷鸣、狂风 | ✅ |
| 洞穴 | 水滴声、回声 | ✅ |
| 城镇 | 人群嘈杂、叫卖声 | ✅ |

### 战斗音效

| 音效 | 说明 |
|------|------|
| 符箓绘制 | 毛笔划过纸面的"唰唰"声 |
| 符箓释放 | 五行属性各有特色（火→爆燃、水→水流、雷→噼啪） |
| 古篆文激活 | 低沉共鸣声 + 能量嗡鸣 |
| 剑诀挥砍 | 破空声 + 命中声 |
| 受击 | 闷响（因伤害类型而异） |
| 闪避 | 风声 + 瞬移音 |
| 格挡 | 金属碰撞声 |
| 暴击命中 | 加强版命中 + 短暂停顿音 |
| 敌人死亡 | 消散声 |

### UI 音效

| 音效 | 触发时机 |
|------|----------|
| 按钮悬停 | 短暂"嘀"声 |
| 按钮点击 | 清脆"咔嗒"声 |
| 打开菜单 | 翻页声 |
| 关闭菜单 | 合拢声 |
| 获得物品 | 拾取"叮"声 |
| 任务完成 | 上升音阶提示音 |
| 境界突破 | 宏大钟声 + 能量爆发 |
| 错误操作 | 低沉"嗡"声 |

---

## 音效格式规范

| 参数 | 值 |
|------|-----|
| 格式 | OGG Vorbis（短音效可用 WAV） |
| 采样率 | 44100 Hz |
| 位深 | 16-bit |
| 短音效时长 | ≤ 3s |
| 环境音循环 | 10~60s 循环 |
