# 音乐设计

> 本章定义游戏的 BGM 风格与场景配乐表。

---

## 音乐风格

| 定位 | 说明 |
|------|------|
| **整体风格** | 东方韵味 + 像素电子 + 氛围音乐 |
| **灵感参考** | 《天涯明月刀》OST、 《仙剑奇侠传》、 东方 Project 编曲 |
| **乐器选择** | 笛子、古筝、琵琶、二胡 + 电子合成器底垫 |
| **节奏** | 探索时舒缓（60~80 BPM），战斗时激昂（120~160 BPM） |

---

## 场景配乐表

| 场景 | 音乐风格 | BPM | 情绪 |
|------|----------|-----|------|
| 标题画面 | 空灵、悠远 | 60 | 宁静、神秘 |
| 青竹谷 | 清新、田园 | 70 | 轻松、安心 |
| 灵田 | 舒缓、治愈 | 65 | 放松、满足 |
| 落霞山脉 | 宽广、壮丽 | 75 | 探索欲 |
| 幽暗密林 | 低沉、悬疑 | 80 | 紧张、好奇 |
| 碧波湖泽 | 流水般柔和 | 70 | 平静、忧郁 |
| 炎火深渊 | 激烈、紧迫 | 130 | 危险、热血 |
| 万雷天谷 | 宏大、史诗 | 140 | 壮烈、决战 |
| 城镇/宗门 | 轻松、热闹 | 80 | 社交、活力 |

### 战斗音乐

| 战斗类型 | 音乐 | BPM |
|----------|------|-----|
| 普通敌人 | 当前场景 BGM 变奏版（增加节奏） | +20 |
| 精英敌人 | 独立战斗曲 | 120~140 |
| Boss 战 | 专属主题曲 | 140~160 |
| 天劫 Boss | 史诗级交响 + 电音 | 160~180 |

---

## 音乐文件规范

| 参数 | 值 |
|------|-----|
| 格式 | OGG Vorbis |
| 采样率 | 44100 Hz |
| 位深 | 16-bit |
| 曲长 | 2~4 分钟（可循环） |
| 交叉淡入淡出 | 场景切换时 2s 过渡 |
