# 音频资源清单

> 本章列出所有音频资源的完整清单与命名规范。

---

## 命名规范

**路径模式：** `res://assets/audio/{type}/{context}_{name}.ogg`

| 类型前缀 | 说明 |
|---------|------|
| `bgm_` | 背景音乐 |
| `sfx_env_` | 环境音效 |
| `sfx_battle_` | 战斗音效 |
| `sfx_ui_` | 界面音效 |
| `sfx_item_` | 物品音效 |
| `sfx_vo_` | 人声/语音 |
