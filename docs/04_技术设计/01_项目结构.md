# 项目结构

> 本章定义 Godot 项目的目录结构与组织规范。

---

## Godot 项目目录树（草案）

```
CloudGrainRecord/
├── project.godot
├── default_bus_layout.tres
│
├── scenes/                    # 场景文件
│   ├── world/                 # 世界/地图场景
│   ├── player/                # 玩家场景
│   ├── enemies/               # 敌人场景
│   ├── items/                 # 物品场景
│   ├── ui/                    # UI 场景
│   └── effects/               # 特效场景
│
├── scripts/                   # 脚本文件
│   ├── autoload/              # Autoload 单例脚本
│   │   ├── signal_bus.gd
│   │   ├── game_manager.gd
│   │   └── settings_manager.gd
│   ├── player/
│   ├── enemies/
│   ├── systems/               # 战斗/符箓/古篆文等系统
│   │   ├── talisman_system.gd
│   │   ├── seal_script_system.gd
│   │   ├── combat_system.gd
│   │   └── block_controller.gd   # 2.5D 方块系统控制器
│   ├── ui/
│   └── utils/                 # 工具函数
│
├── assets/                    # 资源文件
│   ├── sprites/               # 像素精灵
│   ├── blocks/                # 2.5D 方块纹理（预渲染 32×32+ 精灵）
│   │   ├── common/            # 通用方块（grass, dirt, stone, plank...）
│   │   ├── valley/            # 青竹谷场景专属方块
│   │   ├── mountain/          # 落霞山脉场景专属方块
│   │   ├── forest/            # 幽暗密林场景专属方块
│   │   ├── lake/              # 碧波湖泽场景专属方块
│   │   ├── lava/              # 炎火深渊场景专属方块
│   │   ├── thunder/           # 万雷天谷场景专属方块
│   │   └── deco/              # 装饰精灵（花草碎石等）
│   ├── tilesets/              # 图块集（旧系统，逐步迁移到 blocks/）
│   ├── fonts/                 # 字体
│   ├── audio/                 # 音频
│   └── placeholder/           # 占位资源
│
├── resources/                 # 资源定义
│   ├── items/                 # 物品数据
│   ├── talismans/             # 符箓数据
│   ├── seal_scripts/          # 古篆文数据
│   └── recipes/               # 配方数据
│
├── ui/                        # UI 相关
│   ├── themes/                # 主题
│   └── components/            # UI 组件
│
└── addons/                    # 插件
```

## 命名规范

| 类型 | 规范 | 示例 |
|------|------|------|
| 场景文件 | `snake_case.tscn` | `player.tscn`, `fire_talisman.tscn` |
| 脚本文件 | `snake_case.gd` | `player.gd`, `signal_bus.gd` |
| 场景根节点 | PascalCase | `Player`, `FireTalisman` |
| 子节点 | PascalCase | `Sprite2D`, `HitboxArea` |
| 函数 | `snake_case()` | `draw_talisman()` |
| 变量 | `snake_case` | `current_hp` |
| 常量 | `UPPER_SNAKE_CASE` | `MAX_TALISMAN_SLOTS` |
| 信号 | `snake_case` | `talisman_cast` |
| 资源文件 | `snake_case.tres` | `iron_sword_stats.tres` |
| **方块纹理** | `snake_case.png` | `blocks_valley_grass_01.png` |
