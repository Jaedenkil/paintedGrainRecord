# 输入控制

> 本章定义所有输入动作的映射与处理规范。

---

## 设计原则

- **所有输入必须通过 InputMap**，禁止硬编码键位
- 输入处理集中到 **Controller 层**，不分散在单个节点中
- 支持 **键盘 + 鼠标** 和 **手柄** 两种控制方案

---

## 输入处理流程

```
玩家操作
   ↓
InputMap 捕获（动作名）
   ↓
PlayerController._input(event)
   ↓
根据当前状态机状态分发给：
  ├─ MovementHandler    (移动/闪避)
  ├─ CombatHandler      (攻击/符箓/技能)
  └─ UIHandler          (菜单/交互)
```

---

## 按钮图标规范

- UI 中以 **动作名** 引用按钮提示，而非具体键位
- 根据当前检测到的输入设备自动切换图标（键盘/手柄）
- 图标资源位于 `res://ui/icons/input/`
