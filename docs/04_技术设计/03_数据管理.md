# 数据管理

> 本章定义存档系统、数据持久化与配置管理。

---

## 存档系统

### 存档架构

```
GameManager (单例)
   ├── save_data: SaveData (资源对象)
   │     ├── player_data: PlayerData
   │     ├── world_data: WorldData
   │     ├── field_data: FieldData
   │     ├── inventory_data: InventoryData
   │     ├── quest_data: QuestData
   │     └── progression_data: ProgressionData
   │
   └── methods:
         save_game(slot: int) -> void
         load_game(slot: int) -> void
         auto_save() -> void
```

### 存档槽位

| 槽位 | 类型 | 说明 |
|------|------|------|
| 1~5 | 手动存档 | 玩家主动保存 |
| 自动 | 自动存档 | 切换场景/关键节点自动保存 |
| 快速 | 快速存档 | `F5` 快速保存，`F9` 快速加载 |
| 回退 | 备份存档 | 保留最近 3 次自动存档 |

### 存档内容

```
SaveData:
  - version: String               # 存档版本号
  - timestamp: int                 # 保存时间
  - play_time: float               # 游戏时长
  - player_data:
      - name: String
      - level: int
      - realm: int                 # 当前境界
      - stats: Dictionary          # 属性
      - position: Vector2          # 位置
      - inventory: Array[ItemData]
      - equipment: Dictionary
      - talismans: Array[TalismanData]
      - seal_scripts: Array[SealScriptData]
  - world_data:
      - unlocked_areas: Array[String]
      - completed_quests: Array[String]
      - world_state: Dictionary    # 世界状态
  - field_data:
      - level: int
      - plots: Array[PlotData]
      - inventory: Array[ItemData]
  - settings: Dictionary           # 设置快照
```

## 数据持久化路径

| 类型 | 路径 | 格式 |
|------|------|------|
| 存档 | `user://saves/slot_{n}.save` | 二进制 (ResourceSaver) |
| 设置 | `user://settings.cfg` | ConfigFile |
| 截图 | `user://screenshots/` | PNG |

## SettingsManager

管理所有可持久化的玩家设置：

| 分类 | 设置项 |
|------|--------|
| 音频 | 主音量、BGM 音量、SFX 音量 |
| 画质 | 窗口模式、分辨率、VSync、像素缩放 |
| 控制 | 键位映射（InputMap 覆盖）、鼠标灵敏度 |
| 游戏 | 语言、自动存档间隔、伤害数字显示 |
