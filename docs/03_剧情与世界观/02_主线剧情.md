# 主线剧情

> 本章概述《云汲仙田录》的故事大纲与章节划分。

---

## 故事大纲

```
序章：无名散修 → 获得《云汲仙田录》→ 初入修仙路
    ↓
第一章：灵田初开 → 青竹谷探索 → 第一次危机
    ↓
第二章：古篆之谜 → 发现第一个古篆文 → 神秘组织
    ↓
第三章：宗门之争 → 选择宗门 → 宗门任务
    ↓
第四章：秘境之门 → 探索上古遗迹 → 天劫浩渺真相
    ↓
第五章：末法之源 → 揭开灵气衰退的秘密
    ↓
第六章：天劫降临 → 最终之战 → 登仙 / 另一种结局
```

---

## 章节详情

### 序章：机缘
> *"一块灵田，半卷残书，便是你在这末法时代的立身之本。"*

- 主角在一个偏僻山谷中醒来，身无分文
- 探索山谷，找到隐藏的石洞
- 获得《云汲仙田录》残卷 + 一块荒废灵田
- 序章 Boss：一只误入山谷的野兽（教学战）
- 解锁：种植、基础符箓

### 第一章：灵田初开
> *"当第一株灵药在灵田中发芽时，你感受到了一股微弱的灵气波动。"*

- 修复灵田，种植第一批灵药
- 探索青竹谷，结识第一个 NPC
- 遭遇第一个真正威胁——盗匪/妖兽
- Boss：青竹谷妖化竹精
- 解锁：炼丹、炼气突破至筑基

### 第二章：古篆之谜
> *"石碑上的文字古老而神秘，你却莫名地能够理解它的含义……"*

- 发现上古遗迹，获得第一个古篆文
- 神秘组织「暗影楼」现身，试图抢夺古篆文
- 主线战斗 + 追逐剧情
- Boss：暗影楼刺客首领
- 解锁：古篆文系统、筑基突破至金丹

### 第三章：宗门之争
> *"三大宗门各怀心思，你的选择将改变局势。"*

- 可选择加入：天剑宗 / 灵符谷 / 万象阁
- 完成宗门试炼 → 正式入门
- 宗门之间的明争暗斗
- Boss：敌对宗门的精英弟子
- 解锁：宗门贡献系统、高级功法

### 第四章：秘境之门
> *"古篆文拼凑出的地图，指向了一个被封印的秘境……"*

- 收集足够古篆文 → 解读秘境入口位置
- 探索上古秘境 → 发现天劫浩渺的真相
- Boss：秘境守护者（上古傀儡）
- 解锁：元婴突破至化神

### 第五章：末法之源
> *"灵气衰退并非天灾，而是……"*（剧透屏蔽）

- 揭示导致末法时代的真正原因
- 面对幕后黑手
- Boss：暗影楼楼主
- 解锁：化神突破至悟道

### 第六章：天劫降临
> *"最后的试炼，是天道对你的审判。"*

- 九重天劫连环战
- 最终抉择：登仙 / 留在人间 / 重塑世界
- 多重结局
- 终局 Boss：天道化身（可选隐藏 Boss）

---

## 分支与结局

| 结局 | 条件 | 描述 |
|------|------|------|
| 登仙成道 | 主线通关 + 正面声望 | 突破天劫，登仙上界 |
| 人间之尊 | 主线通关 + 中立声望 | 留在人间，成为一方霸主 |
| 重塑天地 | 收集全部古篆文 + 隐藏条件 | 以古篆文之力重构天地法则 |
| 陨落 | 主线失败 | 悟道失败，身死道消（Game Over） |
