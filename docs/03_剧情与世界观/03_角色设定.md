# 角色设定

> 本章列出游戏中的主要角色与关键 NPC。

---

## 主角（玩家角色）

| 项目 | 内容 |
|------|------|
| 默认名 | 云逸（可自定义） |
| 身份 | 无名散修 |
| 性格 | 由玩家选择决定（对话选项塑造） |
| 目标 | 突破境界、解开末法之谜、登仙成道 |
| 配音 | —（像素风文本叙事） |

---

## 主要 NPC

### 青竹老人
| 项目 | 内容 |
|------|------|
| 身份 | 青竹谷隐居老修 |
| 定位 | **新手引导者** |
| 性格 | 慈祥、爱唠叨 |
| 作用 | 教导基础修仙知识、灵田种植、赠送第一把符笔 |

### 苏念薇
| 项目 | 内容 |
|------|------|
| 身份 | 灵符谷天才弟子 |
| 定位 | **同伴 / 符箓导师** |
| 性格 | 活泼、好胜、对古篆文有浓厚兴趣 |
| 作用 | 传授中级符箓、与主角一同探索秘境 |

### 牧云岚
| 项目 | 内容 |
|------|------|
| 身份 | 万象阁阁主 |
| 定位 | **情报提供者 / 交易者** |
| 性格 | 神秘、亦正亦邪 |
| 作用 | 提供古篆文线索、出售稀有物品 |

### 谢渊
| 项目 | 内容 |
|------|------|
| 身份 | 天剑宗首席弟子 |
| 定位 | **竞争对手 / 可结盟** |
| 性格 | 冷傲、正直、责任心强 |
| 作用 | 剑诀导师、宗门线关键角色 |

### 暗影楼主（未命名）
| 项目 | 内容 |
|------|------|
| 身份 | 暗影楼首领 |
| 定位 | **主要反派** |
| 性格 | 冷酷、野心勃勃 |
| 作用 | 主线对抗目标 |

---

## 支持角色

| 角色 | 身份 | 功能 |
|------|------|------|
| 药老 | 丹药阁掌柜 | 炼丹教学、出售丹方 |
| 铁无双 | 铁匠铺师傅 | 炼器教学、装备打造/修理 |
| 灵兽真人 | 灵兽驯养师 | 灵兽系统（后续版本） |
| 百晓生 | 情报贩子 | 奇闻轶事、隐藏任务线索 |
