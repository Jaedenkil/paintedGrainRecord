> 本章定义游戏世界的背景设定与天地法则。

## 世界背景

**天地初开，混沌分阴阳。** 远古时期，天地间灵气充沛，万物皆可修行。上古修士参悟天地法则，将力量封印于 **符箓** 与 **古篆文** 之中，开创了修仙文明。

然而 **千年前的天劫浩劫** 打破了平衡——天地灵气急剧衰退，大量功法失传，古篆文散落各地，修仙界进入 **末法时代**。

千年后的今天，灵气开始复苏。你作为一名 **无名散修**，意外获得一本残破的《云汲仙田录》，上面记载了早已失传的灵田培育之法与古篆文释义。你的修仙之路，由此开启……

## 天地法则

> 以下为《云汲仙田录》世界的根本法则体系。完整九层天道哲学参见 [`01_概览/04_天道哲学体系.md`](../01_概览/04_天道哲学体系.md)。

### 九层天道概览

本世界的运行遵从 **一元→二气→三才→四象→五行→六合→七星→八卦→九宫** 九层衍生法则：

| 层序 | 法则 | 核心 | 在游戏中的主要体现 |
|------|------|------|-------------------|
| **一** | 一元·混元 | 万物之始 | 终极境界、混沌篆、隐藏秘境 |
| **二** | 二气·阴阳 | 对立统一 | 昼夜循环、功法阴阳、丹药阴阳 |
| **三** | 三才·天地人 | 三层框架 | 天时+地利+人和影响所有活动效率 |
| **四** | 四象·太阳太阴少阳少阴 | 四维框架 | 四季系统、四灵守护、四基础属性 |
| **五** | 五行·金木水火土 | 元素构成 | 战斗相克、五脏修炼、五色炼丹 |
| **六** | 六合·天地四方 | 空间整体 | 传送阵、储物空间、六合封印 |
| **七** | 七星·日月五曜 | 时间节律 | 七曜周历、炼丹吉日、修炼吉时 |
| **八** | 八卦·乾坤震巽坎离艮兑 | 八种力量 | 八卦符箓、八卦阵、八门机制 |
| **九** | 九宫·洛书定数 | 数理终极 | 九宫炼丹、九宫阵法、九宫秘境 |

### 灵气
- 灵气是九层天道运转的 **能量载体**，自混元而生，分阴阳二气
- 修士吸收灵气 → 转化为自身修为
- 不同区域灵气浓度不同（灵脉、秘境浓度更高），且受七星周期影响
- 灵田需要聚灵阵汇聚灵气才能培育灵药

### 九层法则的运作原则

1. **层层衍生**：高层法则由低层衍生而来（五行是基础，八卦是五行的八种组合，九宫是八卦的数理排布）
2. **渐进显现**：修士从炼气期的阴阳感悟开始，逐层深入，到登仙时与混元合一
3. **万物皆统摄**：从战斗到炼丹，从种植到建筑，无一不受九层法则的统摄
4. **天人感应**：修士的行为会扰动天道，天道的变化也会影响修士（如七曜周期影响炼丹成功率）

### 天劫
- 突破大境界时天道降下的考验，本质是九层法则对修士的 **校验**
- 渡劫成功 → 境界稳固，实力飞跃
- 渡劫失败 → 修为倒退，需重新积累
- 天劫强度与玩家当前实力、业力值相关
- 高阶天劫（金丹以上）会引入 **八卦天象**（如雷劫来自震卦，心魔劫来自兑卦）
