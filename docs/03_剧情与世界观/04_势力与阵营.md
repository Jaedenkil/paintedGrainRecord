# 势力与阵营

> 本章定义游戏中的主要势力和阵营关系。

---

## 主要势力

### 天剑宗
- **定位**：正道领袖，以剑入道
- **理念**：以剑卫道，斩妖除魔
- **特色**：剑诀流派加成
- **加入条件**：筑基期后可选择

### 灵符谷
- **定位**：符箓正宗
- **理念**：符箓通天地，笔墨定乾坤
- **特色**：符箓流派加成
- **加入条件**：筑基期后可选择

### 万象阁
- **定位**：中立商业组织
- **理念**：万物流通，利益至上
- **特色**：交易折扣、情报服务
- **加入条件**：完成特定支线
- **注意**：非传统宗门，更像商业公会

---

## 敌对势力

### 暗影楼
- **定位**：影藏在暗处的神秘组织
- **目标**：收集古篆文，掌控末法时代的真相
- **行为**：抢夺古篆文、暗杀关键人物、操纵妖兽
- **主要成员**：楼主（未知名）、刺客首领、各地分舵

### 妖化兽群
- **定位**：被灵气污染而狂暴的妖兽
- **出现**：灵气异常区域、夜间
- **威胁**：群居、有组织性

## 阵营关系图

```
       天剑宗 ── 同盟 ── 灵符谷
          │                  │
       友好                友好
          │                  │
         ╰── 万象阁 ── 中立 ── 暗影楼
                  │              │
              中立            敌对
                  │              │
                  ╰── 玩家 ──────╯
```

- 玩家选择加入的宗门将影响与其他势力的关系
- 部分任务需要特定声望等级才能接取
- 声望可通过完成任务、捐献物品提升
