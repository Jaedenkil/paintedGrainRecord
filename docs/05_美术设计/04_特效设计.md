# 特效设计

> 本章定义技能特效、粒子系统与视觉反馈。

---

## 特效类型

### 符箓特效

| 符箓 | 特效描述 | 粒子系统 |
|------|----------|----------|
| 火球符 | 橙红色火球拖尾 | 火焰粒子 + 光晕 |
| 水箭符 | 蓝色冰晶箭矢 | 水花粒子 + 冻结纹理 |
| 金刃符 | 金色弧形斩击 | 锋利光效 + 碎片粒子 |
| 木藤符 | 绿色藤蔓从地面冒出 | 藤蔓生长动画 |
| 土盾符 | 棕黄色护盾光环 | 岩石碎片环绕 |

### 组合符箓特效

| 组合 | 特效描述 |
|------|----------|
| 雷瀑术 | 屏幕震动 + 巨大雷柱从天而降 |
| 熔岩术 | 地面裂开 → 熔岩喷涌 |
| 冰晶术 | 以玩家为中心扩散冰环 |
| 风暴术 | 龙卷风效果 + 被卷起的物体 |
| 地刺术 | 地面连续突刺 |

### 古篆文特效

| 古篆字 | 特效 |
|--------|------|
| 天 | 头顶光环，光柱倾泻 |
| 地 | 地面法阵展开，土元素环绕 |
| 龍 | 金色龙形虚影 |
| 鳳 | 火焰凤凰振翅 |
| 時 | 钟表虚影，时间波纹 |

---

## 粒子系统规范

- 使用 Godot `GPUParticles2D`（GPU 粒子）
- 每个粒子特效文件独立存储为 `.tscn` 或 `.res`
- 粒子最大数量控制在 **200 以内**（性能保证）
- 复用特效使用 **对象池** 管理
- 特效颜色遵循五行色板（金白/木绿/水蓝/火红/土棕）

---

## 视觉反馈原则

| 反馈 | 实现方式 |
|------|----------|
| 命中 | 命中闪光 + HitStop（停帧 3~5 帧） |
| 暴击 | 屏幕微震 + 大号伤害数字 |
| 格挡 | 火花 + 击退特效 |
| 闪避 | 残影效果（Trail） |
| 境界突破 | 全屏光效 + 角色轮廓发光 |
| 任务完成 | 屏幕左上角弹出提示动画 |
