# 角色与动画

> 本章定义角色像素规格与动画需求。

---

## 角色规格

| 角色 | 尺寸 | 动画状态数 | 备注 |
|------|------|-----------|------|
| 主角（男） | 32×48 | ~20 | 四方向 |
| 主角（女） | 32×48 | ~20 | 四方向 |
| 村民 NPC | 24×32 | ~6 | 四方向 |
| 精英 NPC | 32×48 | ~10 | 四方向 |
| 小型敌人 | 24×24 ~ 32×32 | ~8 | 四方向 |
| 大型敌人 / Boss | 48×64 ~ 64×96 | ~12 | 四方向 |

## 动画规范

- 所有动画使用 **SpriteSheet**（精灵表）模式
- 帧动画使用 Godot `AnimatedSprite2D` 节点
- 动画命名：`{角色}_{动作}_{方向}`，如 `player_walk_front`
- 每个动画需标注 **Hurtbox/Hitbox 关键帧**（用于战斗判定）
