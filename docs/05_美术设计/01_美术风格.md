# 美术风格

> 本章定义游戏的视觉风格指南。

---

## 像素风格定位

| 维度 | 说明 |
|------|------|
| **分辨率** | 320×180 内部渲染 → 2×/3× 放大至 640×360 / 960×540 |
| **像素尺寸** | 16×16（基础单位），角色 32×32 ~ 48×48 |
| **色板** | 每个场景限制 16~32 色，整体使用柔和低饱和色调 |
| **帧率** | 角色动画 8~12 FPS，特效 15~24 FPS |
| **风格** | 高精度像素（HD Pixel Art），兼顾细节与清晰度 |

## 方块纹理风格指南（2.5D Block System）

> 本指南定义 45° 斜角俯视 2.5D 方块系统中，每个方块的三个可见面（上平面、左垂直面、右垂直面）的像素绘制规范。

### 2.1 像素绘制规范

| 规范项 | 要求 |
|--------|------|
| **光源方向** | 假设光源来自 **左上角**。上平面接受 100% 光照，左垂直面 80%，右垂直面 60% |
| **面间描边** | 三个面之间使用 1px 深色分隔线，颜色为底面色的 -30% 明度或专用描边色 |
| **外轮廓** | 方块整体外轮廓使用 1px 黑色（`#1a1a1a`）描边，确保方块在场景中独立可辨 |
| **纹理图样精度** | 每个面内部保留完整 16×16 精度的纹理细节（如草地的草叶杂点、石板的裂缝） |
| **垂直面纹理延伸** | 高方块（高度 > 1）的垂直面纹理沿 Y 轴无缝平铺，每 16px 为一个重复单元 |
| **变体规则** | 每个方块类型至少 2 个变体（通过像素级微调纹理图案实现），变体之间色值差异 ≤ 10% |

### 2.2 三面颜色公式

给定 **上平面基准色** `base_color`，其他两个面的颜色按以下公式计算：

```
left_color  = base_color × 0.80  (明度降低 20%)
right_color = base_color × 0.60  (明度降低 40%)
outline     = base_color × 0.30  (描边色)
```

**示例 — 草地方块（基准色 `#4a7c59`）：**

| 面 | 色值 | 明度比例 | 色板预览关键词 |
|----|------|---------|---------------|
| 上平面 (Top) | `#4a7c59` | 100% | 翠绿草色 |
| 左垂直面 (Left) | `#3b6347` | 80% | 暗绿泥土 |
| 右垂直面 (Right) | `#2c4a35` | 60% | 深棕泥土 |
| 描边线 | `#16251a` | 30% | 深色轮廓 |

**示例 — 石方块（基准色 `#8d8d8d`）：**

| 面 | 色值 | 明度比例 | 色板预览关键词 |
|----|------|---------|---------------|
| 上平面 (Top) | `#8d8d8d` | 100% | 浅灰石板 |
| 左垂直面 (Left) | `#717171` | 80% | 中灰石壁 |
| 右垂直面 (Right) | `#555555` | 60% | 深灰石壁 |
| 描边线 | `#2a2a2a` | 30% | 深灰轮廓 |

### 2.4 方块纹理绘制步骤

1. **确定基准色**：从上方色板选择场景对应方块类型的上平面基准色
2. **应用纹理图案**：在基准色上叠加纹理细节（杂点/线条/渐变），每个面独立绘制
   - 上平面：叠加地表纹理（草叶点阵、石板裂缝、沙粒点等）
   - 左垂直面：叠加截面纹理（土粒层理、木纹竖线、石纹横线等）
   - 右垂直面：与左面纹理一致，但颜色更暗
3. **面间描边**：在三个面的交界处绘制 1px 描边线
4. **外轮廓**：在方块整体外边界绘制 1px 深色外轮廓
5. **变体生成**：复制基础纹理，对像素图案做 ±2px 偏移/旋转/镜像变换，生成变体

### 2.5 特殊方块纹理规则

| 特殊类型 | 规则 |
|---------|------|
| **水面方块** | 上平面采用 40~50% alpha 半透明，显示底部纹理。垂直面使用渐变透明度（上深下浅）模拟水的深度 |
| **岩浆方块** | 上平面使用动态纹理（流动动画 4 帧循环），自发光效果。垂直面使用暗红色渐变 |
| **发光方块** | 聚灵阵、灵脉等发光方块，上平面叠加发光光晕层（叠加混合模式） |
| **耕田方块** | 上平面绘制田垄线（等距平行线），可根据湿润度改变颜色深浅 |
| **雪地方块** | 垂直面上半部分覆盖白色，与底部泥土渐变过渡 |
