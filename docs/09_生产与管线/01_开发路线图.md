# 开发路线图

> 本章定义项目的开发里程碑与版本规划。

---

## 版本规划

| 阶段 | 版本 | 内容 | 预计周期 |
|------|------|------|----------|
| **原型** | v0.1 | 核心战斗原型、基础移动、符箓系统原型 | 2~3 个月 |
| **Alpha** | v0.2~0.4 | 核心循环完整、灵田系统、炼丹、第一章 | 4~6 个月 |
| **Beta** | v0.5~0.7 | 全部系统完成、全章节剧情、平衡调整 | 3~4 个月 |
| **先行版** | v0.8 | EA 发布（Steam Early Access） | 2~3 个月 |
| **正式版** | v1.0 | 完整主线、多结局、全区域 | 3~4 个月 |
| **更新** | v1.1+ | 新区域、新敌人、新系统、Bug 修复 | 持续 |

---

## 里程碑分解

### Phase 1: 技术原型（v0.1）

- [ ] Godot 项目初始化，搭建目录结构
- [ ] 玩家移动 + 动画状态机
- [ ] 基础战斗系统（近战攻击）
- [ ] 符箓绘制系统原型（UI + 逻辑）
- [ ] 基础敌人 AI（巡逻/追击/攻击）
- [ ] 一个测试场景

### Phase 2: 核心循环 Alpha（v0.2~v0.3）

- [ ] 灵田系统（种植/生长/收获）
- [ ] 炼丹/炼器系统
- [ ] 背包与物品系统
- [ ] 装备系统
- [ ] 境界与升级系统
- [ ] 青竹谷完整场景
- [ ] 序章 + 第一章剧情
- [ ] 存档/读档系统

### Phase 3: 核心系统完善（v0.4~v0.5）

- [ ] 古篆字符文系统
- [ ] 组合符箓连携
- [ ] 任务系统
- [ ] NPC 对话系统
- [ ] 世界地图 + 传送
- [ ] 更多敌人类型
- [ ] 第二章剧情

### Phase 4: 内容扩展（v0.6~v0.7）

- [ ] 全部区域场景
- [ ] 副本系统
- [ ] Boss 战 ×5
- [ ] 第三章~第六章剧情
- [ ] 多结局系统
- [ ] 随机事件系统
- [ ] 平衡性调整

### Phase 5: 打磨与发布（v0.8~v1.0）

- [ ] UI/UX 整体打磨
- [ ] 美术资源替换（占位→正式）
- [ ] 音频集成
- [ ] 性能优化
- [ ] Bug 测试与修复
- [ ] Steam 集成
- [ ] 本地化（英文）
